    Question 9.4.1.

    We already know we can do the following problem using u-substitution, with \(u = \sin(5 x )\)
    \(\displaystyle \int \sin^{2}\mathopen{}\left(5x\right)\cos\mathopen{}\left(5x\right) \,dx \ = \ \)
    We see that \(\displaystyle \int \sin^{2} (5 x) \cos^3(5 x) \,dx \ \) is solved similarly, because
    • the exponent of \(\sin\) is odd.
    • the exponent of \(\cos\) is odd.
    • None of the above
    So we‚Äôll still use u-substitution with \(u = \sin(5 x )\text{,}\) after first rewriting \(\cos^3 (5 x) = \cos^2(5 x) \cos(5 x )\) and using a trig identity.
    Select the trigonometric identity needed to solve this problem.
    • \(\displaystyle \displaystyle \tan^2 u +1 = \sec^2 u\)
    • \(\displaystyle \displaystyle \sin^2 u + \cos^2 u = 1\)
    • \(\displaystyle \displaystyle \cos^2 u = \frac{1+\cos(2 u)}{2}\)
    • \(\displaystyle \displaystyle 1+ \cot^2 u = \csc^2 u\)
    • \(\displaystyle \displaystyle \sin^2 u = \frac{1-\cos(2 u)}{2}\)
    • None of the above
    Finally, put it all together to calculate \(\displaystyle \int \sin^{2}\mathopen{}\left(5x\right)\cos^{3}\mathopen{}\left(5x\right) \,dx \ = \ \)
    Hint.
    The integral involves \(\sin u\) and \(\cos u\) where the exponent of \(\cos u\) is an odd, positive integer, namely, 3. So we factor out a copy of \(\cos(5 x)\) and use the identity \(\sin^2 u + \cos^2 u = 1\) to turn the remaining \(\cos^2(5 x)\) into an expression involving \(\sin^2(5 x)\text{.}\)
    Thus the integral can be rewritten as
    \begin{equation*} \int \sin^{2}\mathopen{}\left(5x\right)\cos^{3}\mathopen{}\left(5x\right) \, dx = \int \sin^{2} (5 x) \cos^2(5 x) \cos(5 x) \, dx \end{equation*}
    \begin{equation*} = \int \sin^{2} (5 x) \bigg(1-\sin^2(5 x)\bigg) \cos(5 x) \, dx \end{equation*}
    and we use u-substitution.
    Letting \(u= \sin (5 x)\) so \(du = 5 \cos (5 x) \, dx\) and thus \(\displaystyle dx = \frac{ du}{5 \cos (5 x)}\text{.}\)
    This turns the integral into
    \begin{equation*} \int u^{2} (1-u^2) \cos(5 x) \, \frac{ du}{5 \cos (5 x)} = \frac{1}{5}\int u^{2} (1-u^2) \, du \end{equation*}
    which can be multiplied out, and then each part integrated by power rule.

    Question 9.4.2.

    Consider the integral \(\displaystyle \int -7\sin^{2}\mathopen{}\left(6x\right) \,dx\text{.}\)
    In order to integrate, the key feature is that
    • the exponent of \(\sin\) is odd.
    • the exponent of \(\cos\) is odd.
    • the exponents of \(\sin\) and \(\cos\) are both even (remember that 0 is even)
    • None of the above
    Then the trigonometric identity needed to solve this problem is
    • \(\displaystyle \displaystyle \sin^2 u = \frac{1-\cos(2 u)}{2}\)
    • \(\displaystyle \displaystyle 1+ \cot^2 u = \csc^2 u\)
    • \(\displaystyle \displaystyle \cos^2 u = \frac{1+\cos(2 u)}{2}\)
    • \(\displaystyle \displaystyle \sin^2 u + \cos^2 u = 1\)
    • \(\displaystyle \displaystyle \tan^2 u +1 = \sec^2 u\)
    • None of the above
    And we use this identity to rewrite the integral as \(\displaystyle \int -7\sin^{2}\mathopen{}\left(6x\right) \, dx = \ \int\) \(dx\)
    And this can be split into two basic integrals and integrated, though I‚Äôm not asking for you to do that here.

    Question 9.4.4.

    Consider the integral \(\displaystyle \int 5\tan^{3}\mathopen{}\left(10x\right)\sec^{3}\mathopen{}\left(10x\right) \,dx\text{.}\)
    In order to integrate, the key feature is that
    • the exponent of \(\sec\) is odd.
    • the exponent of \(\tan\) is odd.
    • the exponent of \(\tan\) is even.
    • the exponent of \(\sec\) is even.
    • None of the above
    Then the trigonometric identity needed to solve this problem is
    • \(\displaystyle \displaystyle 1+ \cot^2 u = \csc^2 u\)
    • \(\displaystyle \displaystyle \cos^2 u = \frac{1+\cos(2 u)}{2}\)
    • \(\displaystyle \displaystyle \sin^2 u + \cos^2 u = 1\)
    • \(\displaystyle \displaystyle \tan^2 u +1 = \sec^2 u\)
    • \(\displaystyle \displaystyle \sin^2 u = \frac{1-\cos(2 u)}{2}\)
    • None of the above
    So to integrate, we would use u-substitution with \(u= \sec(10 x)\text{.}\) But first we‚Äôd have to rewrite the integral in order to have a copy of \(u\)‚Äôs derivative \(\sec(10 x) \tan(10 x)\text{,}\) and use a trigonometric identity to turn the remaining even powers of \(\tan(10 x)\) into an expression of \(\sec^2(10 x)\text{.}\)
