Section 9.1 From Linear to Polynomial Approximation

Motivating Questions

  • A

Early in our study of calculus in Section 1.8, we learned that if a function \(f\) has a derivative at a fixed value \(x=a\text{,}\) when we zoom in on its graph near \((a,f(a))\text{,}\) the function looks linear. Indeed, such a function is differentiable, and we know that near a fixed input value \(a\text{,}\)

\begin{equation*} f(x) \approx L(x) = f(a) + f’(a)(x-a), \end{equation*}

where \(L\) is the tangent line approximation to \(f\) at \(a\text{.}\)

We see in 9.1.1 that near the point of tangency, the approximation is quite good; at the same time, the tangent line and the function itself only agree exactly at the point of tangency, and we need not go far away from the point of tangency to have the difference between \(f\) and \(L\) be substantial. It is natural to ask: can we find a simple function that approximates \(f\) more closely than \(L\) does and for more values of \(x\) near \(a = 0\text{?}\)

Subsection 9.1.1 From Linear to Quadratic Approximation

In Preview Activity **insert xref**, we found that a quadratic function can be used to produce a better approximation to a given function \(f\) near a fixed point of interest. Indeed, this idea is a natural generalization of the tangent line approximation.

We know that for any function \(f\) that has a derivative at \(a=0\text{,}\) it follows that its tangent line approximation at \(a=0\) is

\begin{equation*} L(x) = f(0) + f’(0)(x - 0) \end{equation*}

Moreover, we can observe that the functions \(L\) and \(f\) have two exact values in common. First, their function values agree at the point of tangency: \(L(0) = f(0)\text{.}\) And second, since \(L(x)\) is a linear function whose slope is \(f’(0)\text{,}\) it is also true that their derivative values agree at the point of tangency: \(L’(0) = f’(0)\text{.}\)

To generate a quadratic function that approximates \(f\) near \(a = 0\text{,}\) a natural requirement we could consider is to have this quadratic function not only share the same function value and derivative value as \(f\) at \(a=0\text{,}\) but also -- in order to have the same concavity -- have the same second derivative value at \(a=0\text{.}\) We can state these requirements more formally as follows: we seek a function \(Q(x)\) of the form

\begin{equation*} Q(x) = c_0 + c_1 x + c_2 x^2 \end{equation*}
