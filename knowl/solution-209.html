(A) The function \(f(x)\) is increasing in the intervals where \(f'(x) \ge 0\) : \((3,8)\)
(B) The function \(f(x)\) is concave down in the intervals where \(f'\) is decreasing: \((0,1)\cup(4,5)\cup(6,8)\) .
(C) The function \(f(x)\) has a local minimum at those values of \(x\) where \(f'(x) =0\) and \(f'\) is negative to the left and positive to the right. There is only one such value: \(x = 3\)
(D) The function \(f(x)\) has an inflection point at those values of \(x\) at which the derivative has a local maximum or a local minimum: \(x =1,4,5,6\)
