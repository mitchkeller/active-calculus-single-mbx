Item b.
For \(h(x) = e^{-x^4}\text{,}\) we start with the series for \(f(x) = e^x\) and then substitute \(-x^2\) for \(x\text{.}\) Doing so,
\begin{equation*} f(x) = e^x = 1 + x + \frac{1}{2!}x^2 + \frac{1}{3!}x^3 + \cdots \end{equation*}
and thus
\begin{equation*} h(x) = g(-x^4) = 1 - x^4 + \frac{1}{2!}x^8 - \frac{1}{3!}x^{12} + \cdots\text{.} \end{equation*}
Since the series for \(e^x\) converges for all real numbers \(x\text{,}\) the same will be true when we replace \(x\) with \(-x^2\text{.}\)
