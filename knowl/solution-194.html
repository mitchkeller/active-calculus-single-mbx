Solution 2.8.4.8.1.

SOLUTION
We multiply numerator and denominator by the conjugate radical:
\begin{equation*} \begin{array}{rcl} \displaystyle \lim_{ x \rightarrow \infty } \left( \sqrt{ x^2 + 2} - \sqrt{x^2 - 7} \right) \amp =\amp \displaystyle \lim_{ x \to \infty } \left( \sqrt{x^2 + 2} - \sqrt{x^2 - 7 } \right) \frac{\sqrt{x^2 + 2} + \sqrt{x^2-7}}{\sqrt{ x^2 + 2} + \sqrt{x^2- 7}} \\ \amp =\amp \displaystyle \lim_{ x \to \infty } \frac{(x^2 + 2) - (x^2-7)}{\sqrt{ x^2 + 2} + \sqrt{x^2 - 7}} \\ \amp = \amp \displaystyle \lim_{ x \to \infty } \frac{ 9}{\sqrt{ x^2 + 2} + \sqrt{x^2 - 7 }} \end{array} \end{equation*}
Notice that the denominator of this last expression \(\left(\sqrt{ x^2 + 2} + \sqrt{x^2 - 7 } \right)\) becomes large as \(x \to \infty\) (itâ€™s bigger than \(x\)). So
\begin{equation*} \lim_{ x \rightarrow \infty } \left( \sqrt{ x^2 + 2} - \sqrt{x^2 - 7} \right) = \displaystyle \lim_{ x \to \infty } \frac{ 9}{\sqrt{ x^2 + 2} + \sqrt{x^2 - 7 }} = 0 \end{equation*}
