The text is available in three different formats: HTML, PDF, and print, each of which is available via links on the landing page at activecalculus.org/. The first two formats are free. If you are going to use the book electronically, the best mode is the HTML 11 . The HTML version looks great in any browser, including on a smartphone, and the links are much easier to navigate in HTML than in PDF. Some particular direct suggestions about using the HTML follow among the next few paragraphs; alternatively, you can watch this short video from the author 12 . It is also wise to download and save the PDF, since you can use the PDF offline, while the HTML version requires an internet connection. A print copy costs about $21 via Amazon 13 .
