Solution 7.3.4.3.1.

A. Euler's method with two steps uses \(\Delta x = 0.5\text{,}\) so we have \(y(0) = 4\text{,}\) \(y(0.5) \approx y(0) + 0.5 ( 7 (0) ) = 4\text{,}\) and \(y(1) \approx y(0.5) + 0.5 ( 7 (0.5) ) = 4 + 1.75 = 5.75\text{.}\) Similarly, with four steps \(\Delta x = 0.25\text{,}\) so \(y(0) = 4\text{,}\) \(y(0.25) \approx y(0) + 0.25 ( 7 (0) ) = 4\text{,}\) \(y(0.5) \approx y(0.25) + 0.25 (7 (0.25)) = 4.4375\text{,}\) \(y(0.75) \approx y(0.5) + 0.25 (7 (0.5)) = 5.3125\text{,}\) \(y(1) \approx y(0.75) + 0.25 (7 (0.75)) = 6.625\text{,}\) B. We can just integrate to find the solution to the differential equation: \(y = \frac{7}{2} x^2 + C\text{.}\) For \(y(0) = 4\text{,}\) we must have \(y = \frac{7}{2} x^2 + 4\text{.}\) C. The error in the two Euler approximations is just the difference between the estimates for \(y(1)\) and the exact value. That is, the difference between 5.75 and 7.5, and between 6.625 and 7.5. Thus, taking the magnitude of the differences, Error in Euler with two steps = 1.75, and Error in Euler with four steps = 0.875. D. Since we doubled the number of steps we took, we expect the error to drop by a factor of two (that is, the error in the Euler approximation with two steps should be twice as big as that with four), which it is.
