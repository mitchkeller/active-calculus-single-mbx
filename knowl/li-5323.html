Item 8.1.5.4.b.
We see that \(T_3(0) = f(0)\) implies \(k_0 = -1\text{;}\) \(T_3'(0) = f'(0)\) implies \(k_1 = -1\text{;}\) \(T_3''(0) = f''(0)\) tells us \(2k_2 = \frac{1}{2}\text{,}\) so \(k_2 = \frac{1}{4}\text{;}\) and \(T_3'''(0) = f'''(0)\) shows that \(6k_3 = 2\text{,}\) so \(k_3 = \frac{1}{3}\text{.}\) Thus \(T_3(x) = \frac{1}{3}x^3 + \frac{1}{4}x^2 - 2x - 1\text{.}\)
