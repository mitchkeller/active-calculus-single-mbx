Solution 1.

\(f'(x)\) when \(f(x) = x^4\text{.}\) Using the limit definition of the derivative,
\begin{equation*} \begin{align*} f'(x) \amp = \lim_{h\rightarrow 0 } \frac{(x+h)^4-x^4}{h} \\ \amp = \lim_{h\rightarrow 0 } \frac{x^4+4x^3h+6x^2h^2+4xh^3+h^4 -x^4}{h} \\ \amp = \lim_{h\rightarrow 0 } \frac{4x^3h+6x^2h^2+4xh^3+h^4}{h} \\ \amp = \lim_{h\rightarrow 0 } (4x^3+6x^2h+4xh^2+h^3) \\ \amp = 4x^3 \end{align*} \end{equation*}
Similarly, the derivative of \(x^3\) is \(3x^2\) and the derivative of \(x^2\) is \(2x\text{.}\) Following the pattern, it is reasonable to conjecture that the derivative of \(x^5\) is \(5x^4\text{,}\) the derivative of \(x^{13}\) is \(13x^{12}\text{,}\) and the derivative of \(x^n\) is \(nx^{n-1}\) for any positive integer \(n\text{.}\)
