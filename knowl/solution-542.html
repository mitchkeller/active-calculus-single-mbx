Solution 8.6.4.2.1.

\(\displaystyle c_n = \frac{f^{(n)}(0)}{n!}\text{,}\) e.g. \(\displaystyle c_0 = \frac{f(0)}{0!} = 0\text{.}\) \(\displaystyle c_1 = \frac{f'(0)}{1!}\text{;}\) \(\displaystyle f'(x)=3 \arctan(8 x) + 3 x\frac{1}{(8 x)^2+1}8 = 3 \arctan(8 x) + \frac{24 x}{64 x^2+1}\text{;}\) \(c_1 = 0\text{.}\) \(\displaystyle c_2 = \frac{f''(0)}{2!}\text{;}\) \(\displaystyle f''(x) = \left( 3 \arctan(8 x) + \frac{24 x}{64 x^2+1} \right)'\) \(\displaystyle = \frac{24}{64 x^2+1} + \frac{24(64 x^2+1)-24 x64\cdot 2x}{(64 x^2+1)^2}\) \(\displaystyle = \frac{1536 x^2 + 24}{(64 x^2+1)^2} + \frac{24 - 1536 x^2}{(64 x^2+1)^2} = \frac{48}{(64 x^2+1)^2}\text{;}\) \(\displaystyle c_2 = \frac{48}{2} = 24\text{;}\) and so on. Another way is to use the power series for arctan: \(\displaystyle \arctan(t) = \sum_{n=0}^{\infty} (-1)^n \frac{t^{2n+1}}{2n+1} = t-\frac{t^3}{3}+\frac{t^5}{5}-\frac{t^7}{7}+...\text{.}\) Then \(\displaystyle 3 x \arctan(8 x) = 3 x \sum_{n=0}^\infty (-1)^n \frac{(8 x)^{2n+1}}{2n+1} = \sum_{n=0}^\infty (-1)^n \frac{3\cdot 8^{2n+1}}{2n+1} x^{2n+2}\) \(\displaystyle = 24 x^2 - \frac{1536}{3} x^4 + \frac{98304}{5} x^6 - \frac{6291456}{7} x^7+...\text{.}\) So \(c_0=0\text{,}\) \(c_1=0\text{,}\) \(c_2=24\text{,}\) \(c_3=0\text{,}\) \(c_4=-\frac{1536}{3}=-512\text{...}\) This is an alternating series. It converges if and only if \(\displaystyle \lim_{n\to\infty}\frac{3 \cdot (8 x)^{2n+1}}{2n+1}=0\text{.}\) This limit is equal to 0 if and only if \(|8 x| \le 1\text{,}\) so \(\displaystyle |x|\le \frac{1}{8}\text{.}\) Therefore the radius of convergence is \(\displaystyle \frac{1}{8}\text{.}\)
