Then, if \(h\) is the height of the box, the volume is given by \(x^2h\text{.}\) We need to find an expression for the height \(h\) in terms of \(x\text{.}\) This is where we use our information about the amount of material used in constructing the box. If the base of the box has sides of length \(x\text{,}\) then \(x^2\) square centimeters of material are used to make the base. Therefore, we have only \(1300 - x^2\) square centimeters of material left to make the sides, of which there are four. Each of the sides uses \(hx\) square centimeters of material. Therefore, we get the formula:
\begin{equation*} 1300-x^2 = 4(hx) \Rightarrow h = \frac{1300-x^2}{4x} \end{equation*}
Plugging this into our formula for volume, we can now write out \(v(x)\) as:
\begin{equation*} v(x) = x^2\left(\frac{1300-x^2}{4x}\right) = \frac{{1300}x - x^3}{4} \end{equation*}
Now, we take the derivative of this expression, using the rules for taking derivatives of polynomials, to get \(v'(x) = \frac{1300}{4} - \frac{3}{4}x^2\text{.}\) Setting this equal to 0 will give us the critical points. When solving, remember that this is a real world situation, so we can not have a negative value for \(x\) (which is a length).
\begin{equation*} \begin{array}{rcl} v'(x) \amp = \amp 0 \\ \frac{1300}{4} - \frac{3}{4}x^2 \amp = \amp 0 \\ \frac{3}{4}x^2 \amp = \amp \frac{1300}{4} \\ x^2 \amp = \amp \frac{1300}{3} \\ x \amp = \amp \sqrt{\frac{1300}{3}} \approx 20.82 \\ \end{array} \end{equation*}
