If a continuous function has a relative maximum at \(c\text{,}\) then it is both necessary and sufficient that the function change from being increasing just before \(c\) to decreasing just after \(c\text{.}\) A continuous function has a relative minimum at \(c\) if and only if the function changes from decreasing to increasing at \(c\text{.}\) (See FigureÂ 3.1.5.) There are only two possible ways for these changes in behavior to occur: either \(f'(c) = 0\) or \(f'(c)\) is undefined. Because these values of \(c\) are so important, we call them critical numbers.
