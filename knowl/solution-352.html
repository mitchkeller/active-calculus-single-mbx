Solution 5.3.5.6.1.

\(w = e^{3 x} + 9\text{.}\) Then \(dw = 3\,e^{3 x}\,dx\text{,}\) so that
\begin{equation*} \int\,\frac{e^{3 x}}{9 + e^{3 x}}\,dx = {1\over 3}\int\,{1\over w}\,dw = {1\over 3}\,\ln(|w|) + C = {1\over 3}\,\ln(|e^{3 x} + 9|) + C. \end{equation*}
Because \(e^{3 x} + 9\) is always positive, we can drop the absolute values from the argument of the natural log.
