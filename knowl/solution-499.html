Solution 8.2.4.5.1.

\(f(0) = 2\text{;}\) \(f'(x) = 6 x - 9\text{,}\) so that \(f'(0) = -9\text{;}\) and \(f''(x) = 6\text{,}\) so that \(f''(0) = 6\text{.}\) Thus
\begin{equation*} T_2(x) = 2 - 9 x + {6\over 2!} x^2 = 2 - 9 x + 3 x^2. \end{equation*}
We notice that \(f(x) = T_2(x)\) in this case, which makes sense because \(f(x)\) is a polynomial.
