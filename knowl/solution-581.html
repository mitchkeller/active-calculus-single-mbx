Solution 9.3.1.1.

\(\displaystyle \int \frac{1}{1+x^2} \, dx = \arctan(x)+C\text{,}\) a basic integral rule.
\(\displaystyle \int \frac{x}{1+x^2} \, dx\) can be solved with u-substitution, letting \(u=1+x^2\) so that \(du = 2x \, dx\text{,}\) which turns the integral into \(\displaystyle \frac{1}{2}\int \frac{1}{u} \, du\text{,}\) which we can integrate.
\(\displaystyle \int \frac{2x+3}{1+x^2} \, dx\) The substitution from the previous problem doesn’t cancel out the \(2x+3\text{.}\) We could split the integral up into \(\displaystyle \int \frac{2x}{1+x^2} \, dx + \int \frac{3}{1+x^2} \, dx\text{,}\) and the first is u-sub, while the second is a basic integral involving \(\arctan (x)\) OR, we could do integration by parts twice. The first time, let \(u=2x+3\) while \(dv = \frac{1}{1+x^2} \, dx\text{,}\) which gets us an answer involving \(\displaystyle \int \arctan(x) \, dx\text{,}\) which can itself be done with integration by parts, letting \(u= \arctan(x)\) and \(dv=dx\text{,}\) giving an integral of \(\displaystyle \int \frac{x}{1+x^2} \, dx\text{,}\) which is solved with u-sub.
\(\displaystyle \int \frac{e^x}{1+(e^x)^2} \, dx\) can be solved with u-substitution, letting \(u=e^x\) so that \(du = e^x \, dx\text{,}\) which turns the integral into \(\displaystyle \int \frac{1}{1+u^2} \, du\text{,}\) which integrates to \(\arctan(u)\text{.}\)
\(\displaystyle \int \frac{1}{x^2-1} \, dx\) can’t be done with u-sub or integration by parts, and it’s not basic.
