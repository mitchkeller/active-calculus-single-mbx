Solution 5.6.6.1.1.

(a) \(\int_{0}^{6}\,e^x\,dx = e^x\bigg|_{0}^{6} = e^6 - e^0 \approx 402.4288\ldots\) (b) Computing the sums directly, since \(\Delta x=3\text{,}\) we have LEFT(2) \(= 3\cdot e^0 + 3\cdot e^{3} \approx 63.2567\) RIGHT(2) \(= 3\cdot e^{3} + 3\cdot e^{6} \approx 1270.54\) TRAP(2) \(= \frac 1 2 ( 63.2567 + 1270.54 ) \approx 666.9\) MID(2) \(= 3\cdot e^{1.5} + 3\cdot e^{3\cdot 1.5} \approx 283.497\) and SIMP(2) = (2 MID(2) + TRAP(2))/3 \(\approx 411.297\text{.}\) And the errors are just the actual value minus the approximate values, or error(LEFT(2)) = 339.172; error(RIGHT(2)) = -868.111; error(TRAP(2)) = -264.471; error(MID(2)) = 118.933; and error(SIMP(2)) = -8.868. (c) Similarly, since \(\Delta x = 0.75\text{,}\) we have LEFT(4) = 173.377; error = 229.052 RIGHT(4) = 777.02; error = -374.591 TRAP(4) = 475.198; error = -72.769 MID(4) = 367.038; error = 35.391 SIMP(4) = 403.091; error = -0.662. (d) For LEFT and RIGHT, we expect the error to go down by 1/2, and this is very roughly what we see. For MID and TRAP, we expect the error to go down by 1/4, and this is approximately what we see. For SIMP, we expect the error to go down by 1/16, again as observed.
