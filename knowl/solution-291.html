Solution 5.1.5.2.1.

\(x=3\text{,}\) we are given that \(f(3)=0\text{.}\) Moving to the left on the interval \(2 \lt x \lt 3\text{,}\) we have \(f'(x) = -2\text{,}\) and we can find \(f(2)\) by knowing that the change in \(f(x)\) over this interval is equal to the area under \(f'(x)\) between \(x=2\) and \(x=3\text{.}\) Thus \(f(3) - f(2) = (-2)(1)\text{,}\) so that \(f(2) = f(3) - (1)(-2) = 2\text{.}\) Similarly, continuing back to \(x=0\text{,}\)
\begin{equation*} f(0) = f(2) - (2)(2) = -2 \end{equation*}
Moving to the right from \(x=3\text{,}\) we know that \(f'(x) = 3\) on \(3 \lt x \lt 5\text{.}\) So, as before, \(f(5) = f(3) + (2)(3) = 6\text{.}\) Similarly, continuing to \(x=7\text{,}\)
\begin{equation*} f(7) = f(5) + (1)(-3) + (1)(1) = 4. \end{equation*}
Then, the value of the integral \(\int_0^7 f'(x)\,dx\) is given by the sum
\begin{equation*} \int_0^7 f'(x)\,dx = (2)(2) + (-2)(1) + (3)(2) + (-3)(1) + (1)(1) = 6. \end{equation*}
Alternatively, knowing \(f(7)=4\) and \(f(0)=-2\) and using the Fundamental Theorem of Calculus, we have
\begin{equation*} \int_0^7 f'(x)\,dx=f(7)-f(0) = (4) - (-2) = 6. \end{equation*}
