Solution 5.3.5.2.1.

\(\int\,7 x \cos(x^{2})\,dx\text{.}\) We use substitution with \(w = x^{2}\) to find this. We have \(dw = 2\,x\,dx\text{,}\) so that
\begin{equation*} \int\,7 x \cos(x^{2})\,dx = {7\over 2}\int\,\cos(w)\,dw = {7\over 2}\,\sin(w) + C = {7\over 2}\,\sin(x^{2}) + C. \end{equation*}
