Solution 1.7.5.4.1.

\(k\text{,}\) the function is continuous at every point except \(x=2\text{.}\) We choose \(k\) to make the function continuous at \(x=2\text{.}\) Since \(5 x^2\) takes the value \(5(2^2)=20\) at \(x=2\text{,}\) we choose \(k\) so that \(kx\) goes through the point \((2,20)\text{.}\) Thus \(k=10\text{.}\)
