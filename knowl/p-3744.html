  1. For \(s(t) = 3\sin(2(t-\frac{\pi}{6})) + 5\text{,}\) \(s'(t) = 3\cos(2(t-\frac{\pi}{6})) \cdot 2 = 6\cos(2(t-\frac{\pi}{6}))\text{.}\) It follows that \(s'(t) = 0\) if and only if \(\cos(2(t-\frac{\pi}{6})) = 0\text{,}\) and thus \(2(t-\frac{\pi}{6}) = \frac{\pi}{2} \pm k\pi\) for some integer value of \(k\text{.}\) Solving this most recent equation for \(t\text{,}\) we have \(t-\frac{\pi}{6} = \frac{\pi}{4} \pm \frac{k\pi}{2}\text{,}\) so
    \begin{equation*} t = \frac{\pi}{6} + \frac{\pi}{4} \pm \frac{k\pi}{2} = \frac{5\pi}{12} \pm \frac{k\pi}{2}\text{.} \end{equation*}
    Now we consider the given interval \([\frac{\pi}{6}, \frac{7\pi}{6}]\text{,}\) which we can think of as \([\frac{2\pi}{12}, \frac{14\pi}{12}]\text{.}\) The value \(t = \frac{5\pi}{12}\) lies in this interval, as does \(t = \frac{5\pi}{12} + \frac{\pi}{2} = \frac{11\pi}{12}\text{.}\) Thus, we have two critical numbers in the interval at which to evaluate \(s\text{,}\) as well as at the endpoints.
    Doing so, we find:
    • \(s(\frac{5\pi}{12}) = 8\text{;}\)
    • \(s(\frac{11\pi}{12}) = 2\text{;}\)
    • \(s(\frac{\pi}{6}) = 5\text{;}\) and
    • \(s(\frac{7\pi}{6}) = 5\text{.}\)
    The absolute max of \(s\) on \([\frac{\pi}{6}, \frac{7\pi}{6}]\) is thus \(s(\frac{5\pi}{12}) = 8\text{,}\) while the absolute min is \(s(\frac{11\pi}{12}) = 2\text{.}\)
  2. From our work in (a), we know that \(s\) has critical numbers at \(t = \frac{5\pi}{12} \pm \frac{k\pi}{2}\text{.}\) On the interval \([0, \frac{\pi}{2}]\text{,}\) only \(t = \frac{5\pi}{12}\) lies in the interval. Thus we have three values at which to evaluate \(s\text{.}\)
    Doing so:
    • \(s(\frac{5\pi}{12}) = 8\text{;}\)
    • \(s(0) = 3\sin(2(0-\frac{\pi}{6})) + 5 = 3\sin(-\frac{\pi}{3}) + 5 = 5 - \frac{3\sqrt{3}}{2} \approx 2.402\text{;}\) and
    • \(s(\frac{\pi}{2}) = 3\sin(2(\frac{\pi}{2}-\frac{\pi}{6})) + 5 = 3\sin(\frac{2\pi}{3}) + 5 = 5 + \frac{3\sqrt{3}}{2} \approx 7.598\text{.}\)
    The absolute max of \(s\) on \([0, \frac{\pi}{2}]\) is thus \(s(\frac{5\pi}{12}) = 8\text{,}\) while the absolute min is \(s(0) = 5 - \frac{3\sqrt{3}}{2} \approx 2.402\text{.}\)
  3. On the interval from \(t = 0\) to \(t = 2\pi\text{,}\) \(s\) completes nearly two full periods, and thus achieves each of its maximum and minimum values. The absolute maximum is \(8\) as determined in (a), and the absolute minimum is \(2\text{.}\) These occur, for example, at \(s(\frac{5\pi}{12}) = 8\text{,}\) while the absolute min is \(s(\frac{11\pi}{12}) = 2\text{.}\) There are other points at which the function achieves these values on the given interval.
  4. On the interval \([\frac{\pi}{3}, \frac{5\pi}{6}] = [\frac{4\pi}{12}, \frac{10\pi}{12}]\text{,}\) only the critical number \(t = \frac{5\pi}{12}\) lies in the interval. Thus we have three values at which to evaluate \(s\text{.}\)
    Doing so:
    • \(s(\frac{5\pi}{12}) = 8\text{;}\)
    • \(s(\frac{\pi}{3}) = 3\sin(2(\frac{\pi}{3}-\frac{\pi}{6})) + 5 = 3\sin(\frac{\pi}{3}) + 5 = \frac{3\sqrt{3}}{2} + 5 \approx 7.598\text{;}\) and
    • \(s(\frac{5\pi}{6}) = 3\sin(2(\frac{5\pi}{6}-\frac{\pi}{6})) + 5 = 3\sin(\frac{4\pi}{3}) + 5 = 5 - \frac{3\sqrt{3}}{2} \approx 2.402\text{.}\)
    The absolute max of \(s\) on \([\frac{\pi}{3}, \frac{5\pi}{6}]\) is thus \(s(\frac{5\pi}{12}) = 8\text{,}\) while the absolute min is \(s(\frac{5\pi}{6}) \approx 2.402\text{.}\)
