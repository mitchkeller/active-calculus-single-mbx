The graph of \(f\) is a parabola that will open up if \(a>0\text{,}\) in which case it has a global minimum at \((h,k)\text{.}\) The derivative is \(f'(x)=2a(x-h)\text{,}\) so there is a critical value at \(x=h\text{.}\) It is easy to use a derivative sign chart to confirm that this critical value yields a maximum if \(a\lt 0\text{.}\)
