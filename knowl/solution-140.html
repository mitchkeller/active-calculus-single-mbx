Solution 2.4.3.1.

  1. \(p(x) = \csc(x)\) is defined for all \(x\) for which \(\sin(x) \ne 0\text{.}\) Hence the domain of \(h\) is all real numbers \(x\) such that \(x \ne k\pi\text{,}\) where \(k = 0, \pm 1, \pm 2, \ldots\text{.}\)
  2. By the quotient rule,
    \begin{equation*} h'(x) = \frac{0 - 1 \cdot (\cos(x))}{\sin^2(x)} = -\frac{\cos(x)}{\sin^2(x)}\text{.} \end{equation*}
  3. Observe that \(h'(x) = -\frac{\cos(x)}{\sin^2(x)} = -\frac{1}{\sin(x)} \cdot \frac{\cos(x)}{\sin(x)}\text{,}\) so
    \begin{equation*} h'(x) = -\csc(x) \cot(x)\text{.} \end{equation*}
  4. The derivative \(p'(x)\) is, like \(p(x)\text{,}\) defined for all values of \(x\) for which \(\sin(x) \ne 0\text{.}\) Therefore, \(p\) and \(p'\) have the same domain: all real numbers \(x\) such that \(x \ne k\pi\text{,}\) where \(k = 0, \pm 1, \pm 2, \ldots\text{.}\)
