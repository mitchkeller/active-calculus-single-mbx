Item b.
Let \(h(x) = \ln(1+x)\text{.}\) We use substitution and integration to develop the Taylor series of \(h(x)\) in a different way.
  1. We know that \(p(u) = \frac{1}{1-u} = 1 + u + u^2 + u^3 + \cdots\) and this series converges for \(|u| \lt 1\text{.}\)
  2. Letting \(u = -x\text{,}\) we have
    \begin{equation*} r(x) = \frac{1}{1+x} = p(-x) = 1 - x + x^2 - x^3 + \cdots\text{.} \end{equation*}
  3. Finally, since
    \begin{equation*} \ln(1 + x) = \int_0^x \frac{1}{1+t} \, dt\text{,} \end{equation*}
    it follows that
    \begin{align*} \ln(1 + x) =\mathstrut \amp \int_0^x \frac{1}{1+t} \, dt\\ =\mathstrut \amp \int_0^x 1 - t + t^2 - t^3 + \cdots \, dt\\ =\mathstrut \amp \left. t - \frac{1}{2}t^2 + \frac{1}{3} t^3 - \frac{1}{4}t^4 + \cdots \right|_0^x\\ =\mathstrut \amp x - \frac{1}{2}x^2 + \frac{1}{3} x^3 - \frac{1}{4}x^4 + \cdots \end{align*}
    and this series is guaranteed to converge on the same interval as the series we started with (\(-1 \lt x \lt 1\)). This is also the same series we found for \(h(x) = \ln(1+x)\) working directly from the definition of a Taylor series.
