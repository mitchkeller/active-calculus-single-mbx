Solution 8.2.4.10.1.

  1. We have that
    \begin{equation*} \ln(1.5) = f(0.5) \approx T_5(0.5) = 1(0.5) - \frac{1}{2}(0.5)^2 + \frac{1}{3}(0.5)^3 - \frac{1}{4}(0.5)^4 + \frac{1}{5}(0.5)^5 = 0.4072916\overline{6}\text{.} \end{equation*}
  2. Here, we find that
    \begin{equation*} \ln(1.5) = g(1.5) \approx P_5(1.5) = 1(1.5-1) - \frac{1}{2}(1.5-1)^2 + \frac{1}{3}(1.5-1)^3 - \frac{1}{4}(1.5-1)^4 + \frac{1}{5}(1.5-1)^5\text{.} \end{equation*}
    Since \((1.5 - 1) = 0.5\text{,}\) we see that \(P_5(1.5) = T_5(0.5) = 0.4072916\overline{6}\text{.}\)
  3. In (b), we just observed that the estimates of \(\ln(1.5)\) generated by \(T_5(x)\) and \(P_5(x)\) are the same. This occurs because \(f(x) = \ln(1+x)\) is just a shifted version of \(g(x) = \ln(x)\) (\(f(x) = g(1+x)\text{,}\) so \(f\) is \(g\) shifted 1 unit left), so by changing where their respective Taylor polynomials are centered, we are using the same structure to generate each approximation.
