Solution 3.2.3.1.1.

Solution: Use the Quotient Rule to find \(f'\) and \(f''\text{.}\) Set each equal to zero and solve to answer the problem.
\begin{equation*} f'(x) = -\dfrac{2ax}{(x^2 + b^2)^2} = 0 \;\Rightarrow\; x = 0. \end{equation*}
\begin{equation*} f''(x) = \dfrac{6ax^2 -2ab^2}{(x^2 + b^2)^3} = 0 \;\Rightarrow\; x^2 = \dfrac{b^2}{3} \;\Rightarrow\; x = \pm \dfrac{b}{\sqrt{3}}. \end{equation*}
