We recall that a function \(f\) is said to be differentiable at \(x = a\) if \(f'(a)\) exists. Moreover, for \(f'(a)\) to exist, we know that the function \(y = f(x)\) must have a tangent line at the point \((a,f(a))\text{,}\) since \(f'(a)\) is precisely the slope of this line.
