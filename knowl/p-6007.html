For the first integral, \(\int e^{2x} \sin(e^x) \, dx\text{,}\) since \(e^x\) is the inside function in the composition, we let \(z = e^x\text{,}\) so \(dz = e^x \, dx\text{,}\) and write \(e^{2x} = e^x \cdot e^x\text{.}\) Hence,

\begin{equation*} \int e^{2x} \sin(e^x) \, dx = \int e^x \cdot e^x \sin(e^x) \, dx = \int z \sin(z) \, dz\text{.} \end{equation*}

Using the standard integration by parts with \(u = z\) and \(dv = \sin(z) \, dz\text{,}\) we find

\begin{equation*} \int z \sin(z) \, dz = \sin(z) - z\cos(z) + C\text{.} \end{equation*}

Re-substituting from \(z\) back to \(x\text{,}\) we've found that

\begin{equation*} \int e^{2x} \sin(e^x) \, dx = \sin(e^x) - z\cos(e^x) + C\text{.} \end{equation*}
