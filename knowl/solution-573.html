Solution 8.5.5.8.1.

  1. Letting \(x = 0\) in
    \begin{equation*} e^x = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \cdots + a_n x^n + \cdots \end{equation*}
    shows that \(e^0 = a_0\text{,}\) so \(a_0 = 1\text{.}\)
  2. Differentiating both sides of Equation (8.5.5), since \(\frac{d}{dx}[e^x] = e^x\) we find
    \begin{equation*} e^x = a_1 + 2a_2 x + 3a_3 x^2 + \cdots + n a_n x^{n-1} + \cdots\text{.} \end{equation*}
    By equating the constant terms in Equation (8.5.5) and in our most recent equation (Equation 2), we see that
    \begin{equation*} a_0 = a_1\text{.} \end{equation*}
    Having earlier established that \(a_0 = 1\text{,}\) we now also know \(a_1 = 1\text{.}\)
  3. If we equate the coefficients of the linear terms \(a_1 x\) and \(2a_2 x\) in Equation (8.5.5) and Equation 2, we see that \(a_1 = 2a_2\text{,}\) so \(a_2 = \frac{1}{2}a_1 = \frac{1}{2}\text{.}\)
  4. Equating coefficients of like terms in Equation (8.5.5) and Equation 2, we find that \(3a_3 = a_2\text{,}\) \(4a_4 = a_3\text{,}\) and \(5a_5 = a_4\text{.}\) This implies that \(a_3 = \frac{1}{3}a_2 = \frac{1}{3 \cdot 2} = \frac{1}{3!}\text{,}\) \(a_4 = \frac{1}{4}a_3 = \frac{1}{4 \cdot 3!} = \frac{1}{4!}\text{,}\) and \(a_5 = \frac{1}{5}a_4 = \frac{1}{5 \cdot 4!} = \frac{1}{5!}\text{.}\) Thus we observe that \(a_k = \frac{1}{k!}\text{,}\) which is exactly what we’d expect for the Taylor series of \(e^x\text{.}\)
