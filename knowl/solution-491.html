Solution 8.2.2.1.

Let \(f(x) = \cos(x)\text{.}\)
  1. Determine the first \(8\) derivatives of \(f(x) = \cos(x)\) and evaluate each at \(a = 0\text{.}\)
  2. Using the results in Table 8.2.13 along with the fact that \(c_k = \frac{f^{(k)}(0)}{k!}\text{,}\) we find that
    \begin{equation*} T_8(x) = 1 + 0x - \frac{1}{2!}x^2 + 0x^3 + \frac{1}{4!}x^4 + 0x^5 - \frac{1}{6!}x^6 + 0x^7 + \frac{1}{8!}x^8\text{.} \end{equation*}
  3. Based on the pattern in Table 8.2.13, we expect that \(T_{10}(x) = 1 - \frac{1}{2!}x^2 + \frac{1}{4!}x^4 - \frac{1}{6!}x^6 + \frac{1}{8!}x^8 - \frac{1}{10!}x^{10}\text{.}\)
