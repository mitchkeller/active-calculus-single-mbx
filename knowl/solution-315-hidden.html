\(f(x) = 3 x\) between \(x=1\) and \(x = b\) is given by \(\int_1^b (3 x) dx\text{.}\) Using the Fundamental Theorem to evaluate the integral:
\begin{equation*} \hbox{Area } = \frac{3}{2} x^2 \bigg|_1^b = \frac{3}{2} b^2 - \frac{3}{2}. \end{equation*}
Since the area is 12, we have
\begin{equation*} \frac{3}{2} b^2 - \frac{3}{2} = 12, \end{equation*}
so
\begin{equation*} \frac{3}{2} b^2 = \frac{27}{2}, \end{equation*}
and thus
\begin{equation*} b^2 = 9,\quad\hbox{or}\quad b = \pm 3. \end{equation*}
Since \(b\) is larger than 1, we have \(b = 3\text{.}\)
