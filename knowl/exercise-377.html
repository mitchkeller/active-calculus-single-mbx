Exercise 4.

In this exercise, we extend our work in Exercise 8.1.5.3. We continue to consider the function \(f(x) = \frac{1}{3}x^3 + \frac{1}{4}x^2 - 2x - 1\text{,}\) but now build the cubic (degree \(3\)) approximation to \(f\) near \(a = 0\text{.}\)
  1. Let \(T_3(x) = k_0 + k_1x + k_2x^2 + k_3x^3\) and determine \(T_3'(x)\text{,}\) \(T_3''(x)\text{,}\) and \(T_3'''(x)\text{.}\) In addition, determine \(f'''(x)\text{.}\) Record your results in Table 8.1.17, along with the values of each of these functions at \(a = 0\text{.}\)
  2. Use your work in Table 8.1.17 to find the formula for \(T_3(x)\) that satisfies \(T_3(0) = f(0)\text{,}\) \(T_3'(0) = f'(0)\text{,}\) \(T_3''(0) = f''(0)\text{,}\) and \(T_3'''(0) = f'''(0)\text{.}\)
  3. Plot \(f(x)\text{,}\) \(T_1(x)\text{,}\) \(T_2(x)\text{,}\) and \(T_3(x)\) on the same axes, centered at \(a = 0\text{.}\) What do you notice?
  4. What do you expect the degree \(4\) approximation to \(f(x)\text{,}\) \(T_4(x)\) to be? Why?
  5. What do you expect will happen if we find the degree \(6\) approximation to any function \(f(x)\) that is itself a degree \(6\) polynomial?
