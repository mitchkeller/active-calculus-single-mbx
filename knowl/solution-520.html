Solution 8.5.6.1.1.

\(f(x) = \sin\!\left(x\right)\) Then \(f(0) = \sin(0) = 0\text{,}\) and
\begin{equation*} f'(x) = \cos\!\left(x\right),\qquad \hbox{so}\quad f'(0) = 1, \end{equation*}
\begin{equation*} f''(x) = -\sin\!\left(x\right),\qquad \hbox{so}\quad f''(0) = 0, \end{equation*}
\begin{equation*} f'''(x) = -\cos\!\left(x\right),\qquad \hbox{so}\quad f'''(0) = -1, \end{equation*}
\begin{equation*} f^{4}(x) = \sin\!\left(x\right),\qquad \hbox{so}\quad f^{4}(0) = 0, \end{equation*}
\begin{equation*} f^{5}(x) = \cos\!\left(x\right),\qquad \hbox{so}\quad f^{5}(0) = 1, \end{equation*}
Thus,
\begin{equation*} P_{1}(x) = x, \end{equation*}
\begin{equation*} P_{3}(x) = x - {1\over6} x^3, \end{equation*}
and
\begin{equation*} P_{5}(x) = x - {1\over6} x^3 + {1\over120} x^5. \end{equation*}
