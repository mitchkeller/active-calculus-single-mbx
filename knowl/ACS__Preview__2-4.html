Preview Activity 2.4.1.

Consider the function \(f(x) = \tan(x)\text{,}\) and remember that \(\tan(x) = \displaystyle{\frac{\sin(x)}{\cos(x)}}\text{.}\)
(a) What is the domain of \(f\text{?}\)
  • A. All positive real numbers.
  • B. All real numbers except zero
  • C. All real numbers except \(0, \pm\pi, \pm 2\pi, \pm 3\pi, \ldots\)
  • D. All real numbers except \(\pm \frac{\pi}{2}, \pm \frac{3\pi}{2}, \pm \frac{5\pi}{2}, \ldots\)
  • E. All real numbers
  • F. None of these
(b) Compute \(f'(x)\) using the quotient rule. Which of the following is a possible correct answer?
  • A. \(\displaystyle{f'(x)=\frac{\cos(x)\sin(x)+\sin(x)\cos(x)}{\cos^2(x)}} \)
  • B. \(\displaystyle{f'(x)=\frac{\sin(x)\sin(x)-\cos(x)\cos(x)}{\cos^2(x)}} \)
  • C. \(\displaystyle{f'(x)=\frac{\cos(x)\cos(x)-\sin(x)\sin(x)}{\cos^2(x)}} \)
  • D. \(\displaystyle{f'(x)=\frac{\cos(x)\sin(x)-\sin(x)\cos(x)}{\cos^2(x)}} \)
  • E. \(\displaystyle{f'(x)=\frac{\cos(x)\cos(x)+\sin(x)\sin(x)}{\cos^2(x)}} \)
  • F. None of these
(c) Simplify the correct answer to part (b) using the Fundamental Trigonometric Identity. Which of the following is a correct answer?
  • A. \(\displaystyle{f'(x)=\frac{\cos(x)-\sin(x)}{\cos^2(x)}} \)
  • B. \(\displaystyle{f'(x)=\frac{\sin(x)-\cos(x)}{\cos^2(x)}} \)
  • C. \(\displaystyle{f'(x)=\frac{1}{\cos^2(x)}} \)
  • D. \(\displaystyle{f'(x)=\frac{\cos(2x)\sin(2x)}{\cos^2(x)}} \)
  • E. \(\displaystyle{f'(x)=\frac{2\cos(x)\sin(x)}{\cos^2(x)}} \)
  • F. None of these
(d) Simplify the correct answer to part (c) using another trigonometric identity. Which of the following is a correct answer?
  • A. \(f'(x) = \cot(x) \)
  • B. \(f'(x) = \cot^2(x) \)
  • C. \(f'(x) = \sec(x) \)
  • D. \(f'(x) = \csc^2(x) \)
  • E. \(f'(x) = \csc(x) \)
  • F. \(f'(x) = \sec^2(x) \)
  • G. None of these
(e) For what values of \(x\) is \(f'(x)\) defined?
  • A. All positive real numbers.
  • B. All real numbers except zero
  • C. All real numbers except \(0, \pm\pi, \pm 2\pi, \pm 3\pi, \ldots\)
  • D. All real numbers except \(\pm \frac{\pi}{2}, \pm \frac{3\pi}{2}, \pm \frac{5\pi}{2}, \ldots\)
  • E. All real numbers
  • F. None of these
