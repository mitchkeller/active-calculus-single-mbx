Item 8.3.5.7.a.
We consider the finite geometric series that results when \(a = 1\) and \(r = 1\text{:}\)
\begin{equation*} S_n = 1 + 1 + \cdots + 1\text{,} \end{equation*}
which has \(n\) terms in the sum.
  1. Using
    \begin{equation*} S_n = 1 + 1 + \cdots + 1\text{,} \end{equation*}
    we see that \(S_2 = 1 + 1 = 2\text{,}\) \(S_3 = 1 + 1 + 1 = 3\text{,}\) \(S_4 = 4\text{,}\) and more generally, \(S_n = n\text{.}\)
  2. Because the sequence \(1, 2, 3, 4, \ldots, n, \ldots\) increases without bound, partial sums of the series never approach a single finite value, so the sequence diverges, which means the infinite sum \(S = 1 + 1 + \cdots + 1 + \cdots\) also diverges.
