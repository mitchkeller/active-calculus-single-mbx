Similarly,
\begin{equation*} f(2) - f(0) = \int_0^2 f'(x) \, dx, \end{equation*}
and using geometry, remembering that below the x-axis counts negatively, we see that \(\int_0^2 f'(x) \, dx = \frac{1}{2}\text{,}\) and so \(f(2) = 1.5\text{.}\) And continue in this way to calculate that \(f(3) = -0.5\text{,}\) \(f(4) = -2\text{,}\) \(f(5) = -0.5\text{,}\) \(f(6) = 1\text{.}\)
