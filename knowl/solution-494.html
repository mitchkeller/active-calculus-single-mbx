Solution 7.4.3.4.1.

\begin{equation*} \frac{dy}{y^2} = (5 + t)\,dt, \end{equation*}
so
\begin{equation*} \int \frac{dy}{y^2} = \int (5 + t)\,dt, \end{equation*}
and
\begin{equation*} -\frac{1}{y} = {1\over2}(5 + t)^2 + C, \end{equation*}
giving
\begin{equation*} y=-\frac{2}{(5 + t)^2 + A}, \end{equation*}
where \(A = 2C\text{.}\) Since \(y=3\) when \(t=1\text{,}\) we have
\begin{equation*} 3 = -\frac{2}{6^2 + A},\quad \hbox{ so } \quad 3 A + 108 = -2, \end{equation*}
and \(A = -{{110\over3}}\text{.}\) Thus
\begin{equation*} y = -\frac{2}{(5 + t)^2 - {{110\over3}}}. \end{equation*}
