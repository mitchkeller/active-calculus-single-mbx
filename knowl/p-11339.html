We know that \(f'(x) = -\sin(x)\text{,}\) \(f''(x) = -cos(x)\text{,}\) \(f'''(x) = \sin(x)\text{,}\) and \(f^{(4)}(x) = cos(x)\text{,}\) so \(f'\left(\frac{\pi}{2}\right) = -\sin\left(\frac{\pi}{2}\right) = -1\text{,}\) \(f''\left(\frac{\pi}{2}\right) = -cos\left(\frac{\pi}{2}\right) = 0\text{,}\) \(f'''\left(\frac{\pi}{2}\right) = \sin\left(\frac{\pi}{2}\right) = 1\text{,}\) and \(f^{(4)}(x) = cos\left(\frac{\pi}{2}\right) = 0\text{.}\) We also note that \(f\left(\frac{\pi}{2}\right) = \cos\left(\frac{\pi}{2}\right) = 0\)
