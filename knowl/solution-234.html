Solution 3.3.4.2.1.

\begin{equation*} g'(t) = 4\cdot e^{-6 t} - 24 te^{-6 t} = 4 (1 - 6 t)e^{-6 t}, \end{equation*}
so the critical point is \(t=1/6\text{.}\) Since \(g'(t)>0\) for \(0\lt t\lt 1/6\) and \(g'(t)\lt 0\) for \(t>1/6\text{,}\) the critical point is a local maximum. As \(t\to\infty\text{,}\) the value of \(g(t)\to0\text{,}\) and as \(t\to0^+\text{,}\) the value of \(g(t)\to0\text{.}\) Thus, the local maximum at \(t=1/6\) is a global maximum of \(g(1/6)={4\over 6} e^{-1} = {4\over 6 e}\text{.}\) In addition, the value of \(g(t)\) is positive for all \(t>0\text{;}\) it tends to 0 but never reaches 0. Thus, there is no global minimum.
