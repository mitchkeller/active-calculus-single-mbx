\(f'(x)\) we use the product and chain rules: \(f'(x) = a + (1)(\ln(bx)) + x (\frac1{bx}\cdot b) = a + \ln(bx) + 1\text{.}\) For \(x > 1\) we know that \(\ln(bx) > 0\text{,}\) so, given that \(a > 0\text{,}\) \(f'(x) > 0\) and we know that \(f(x)\) must be increasing. Then \(f''(x) = \frac1x > 0\text{,}\) so the function must be concave up.
