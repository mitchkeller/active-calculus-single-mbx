Exercise 7.

Consider the piecewise linear function \(f\) given in Figure 5.1.8. Let the functions \(A\text{,}\) \(B\text{,}\) and \(C\) be defined by the rules \(A(x) = \int_{-1}^{x} f(t) \, dt\text{,}\) \(B(x) = \int_{0}^{x} f(t) \, dt\text{,}\) and \(C(x) = \int_{1}^{x} f(t) \, dt\text{.}\)
  1. For the values \(x = -1, 0, 1, \ldots, 6\text{,}\) make a table that lists corresponding values of \(A(x)\text{,}\) \(B(x)\text{,}\) and \(C(x)\text{.}\)
  2. On the axes provided in Figure 5.1.8, sketch the graphs of \(A\text{,}\) \(B\text{,}\) and \(C\text{.}\)
  3. How are the graphs of \(A\text{,}\) \(B\text{,}\) and \(C\) related?
  4. How would you best describe the relationship between the function \(A\) and the function \(f\text{?}\)
