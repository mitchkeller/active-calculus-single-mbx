2. Any solution curve that starts with \(P(0) \gt 3\) will decrease to \(P(t) = 3\) as \(t \to \infty\text{;}\) any curve that starts with \(1 \lt P(0) \lt 3\) will increase to \(P(t) = 3\text{;}\) any curve that starts with \(0 \lt P(0) \lt 1\) will decrease to \(P(t) = 0\text{.}\)
