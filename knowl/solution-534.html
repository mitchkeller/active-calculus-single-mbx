Solution 8.2.4.7.1.

  1. Using the given information that \(f(0) = 2, f'(0) = -3, f''(0) = -1, f'''(0) = 0, f^{(4)}(0) = -3\) and the fact that the coeffficients of the Taylor polynomial centered at \(a = 0\) are \(c_k = \frac{f^{k}(0)}{k!}\text{,}\) we find that
    \begin{equation*} T_4(x) = 2 - 3x - \frac{1}{2!}x^2 - \frac{3}{4!}x^4\text{.} \end{equation*}
  2. Since \(0.5\) is near \(0\text{,}\) we have
    \begin{equation*} f(0.5) \approx T_4(0.5) = 2 - 3 \cdot 0.5 - \frac{1}{2!}(0.5)^2 - \frac{3}{4!}(0.5)^4 = 0.3671875\text{.} \end{equation*}
  3. Since each lower degree Taylor polynomial is embedded within a higher degree one, now that we know \(T_4(x)\text{,}\) it follows that
    \begin{align*} T_3(x) =\mathstrut \amp 2 - 3x - \frac{1}{2!}x^2 + 0x^3\\ T_2(x) =\mathstrut \amp 2 - 3x - \frac{1}{2!}x^2\\ T_1(x) =\mathstrut \amp 2 - 3x\text{.} \end{align*}
    We note that \(T_3(x) = T_2(x)\) since \(f'''(0) = 0\text{.}\)
