Exercise 7.

Suppose we know the following information about a function \(f\text{:}\)
\begin{equation*} f(0) = 2, f'(0) = -3, f''(0) = -1, f'''(0) = 0, f^{(4)}(0) = -3\text{.} \end{equation*}
  1. Determine \(T_4(x)\text{,}\) the degree \(4\) Taylor polynomial of \(f\) that is centered at \(a = 0\text{.}\)
  2. Use \(T_4(x)\) to estimate \(f(0.5)\text{.}\)
  3. State each of the Taylor polynomials \(T_3(x)\text{,}\) \(T_2(x)\text{,}\) and \(T_1(x)\text{.}\)
