Solution 1.3.3.1.1.

A. Recall that the derivative is negative when the function is decreasing. So we're looking for a point on the graph where the function is decreasing; one such point is \(x = 1\text{.}\) B. The function is negative when the function is below the \(x\)-axis. One such point is \(x = 1.5\text{.}\) C. The derivative of the function is smallest (most negative) when the function is decreasing the fastest. This occurs at \(x \approx 3.25\text{.}\) D. The derivative of the function is zero when the function has a horizontal tangent. One such point is \(x \approx 1.45\text{.}\) E. Finally, we're looking for a point where the derivative is the same as the derivative at \(x = 2.25\text{.}\) This means that we want the slope of a tangent to the curve to be approximately the same as it is at \(x = 2.25\text{.}\) Such a point is \(x = 1.75\text{.}\)
