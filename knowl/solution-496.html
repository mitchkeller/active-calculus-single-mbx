Solution 8.2.4.3.1.

\(x\) = a:
\begin{equation*} T(x) = f(a) + \frac{f'(a)}{1!} (x-a) + \frac{f''(a)}{2!} (x-a)^2 + ... + \frac{f^n(a)}{n!} (x-a)^n \end{equation*}
\begin{equation*} f(x) = \ln\!\left(x+1\right) \end{equation*}
\begin{equation*} f'(x) = \frac{1}{x+1} \end{equation*}
\begin{equation*} f''(x) = \frac{-1}{\left(x+1\right)^{2}} \end{equation*}
\begin{equation*} f'''(x) = \frac{2}{\left(x+1\right)^{3}} \end{equation*}
So, in this case, \(f(x) = \ln\!\left(x+1\right)\) and \(T_2(x) = A + B(x-0) + C(x-0)^2\) where \(A=f(0)=\ln (1)=0\) \(B=f'(0)=\frac{1}{(1)}=1\) \(C=\frac{f''(0)}{2!}=-\frac{1}{(2)}=-0.5\) Thus,
\begin{equation*} T_2(x) = \ln (1) + \frac{1}{(1)} (x - 0) - \frac{1}{(2)} (x - 0)^2 \end{equation*}
or,
\begin{equation*} T_2(x) = x-0.5x^{2} \end{equation*}
Similarly, \(T_3(x) = D + E(x-0) + F(x-0)^2 + G(x-0)^3\) where \(D=f(0)=\ln (1)=0\) \(E=f'(0)=\frac{1}{(1)}=1\) \(F=\frac{f''(0)}{2!}=-\frac{1}{(2)}=-0.5\) \(G=\frac{f'''(0)}{3!}=\frac{1}{(3)}=0.333333\) Thus,
\begin{equation*} T_3(x) = \ln (1) + \frac{1}{(1)} (x - 0) - \frac{1}{(2)} (x - 0)^2 + \frac{1}{(3)} (x - 0)^3 \end{equation*}
or,
\begin{equation*} T_3(x) = T_2(x) + 0.333 (x-0)^3 \end{equation*}
