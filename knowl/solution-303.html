Solution 5.2.5.3.1.

A. The definition of \(g\) gives \(g(2) = \int_{2}^{2} f(t)\,dt = 0\text{.}\) B. The Fundamental Theorem gives \(g'(5) = f(5) = 2.00875764023537\text{.}\) C. The function \(g\) is concave up where \(g''\) is positive. Since \(g''=f'\text{,}\) we see that \(g\) is concave up where \(f\) is increasing. This occurs on the interval \(x\in (1.65,4.59)\text{.}\) D. The function \(g\) increases when \(f\) is positive and decreases when it is negative, so we're looking for the point at which the greatest positive area is accumulated. Thus \(g\) takes its maximum value at \(x=6.22504\text{.}\)
