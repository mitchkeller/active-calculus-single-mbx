To determine how long it will take us to pay off the card, we need to find the time so that \(A(t) = 0\text{.}\) When \(A(t) = 0\) we have

\begin{align*} 0 \amp = \left(1000 - \frac{12(25)}{0.2}\right)\left(1+\frac{0.2}{12}\right)^{12t} + \frac{12(25)}{0.2}\\ 0 \amp = (-500)(1.0167)^{12t} + 1500\\ 500(1.0167)^{12t} \amp = 1500\\ (1.0167)^{12t} \amp = 3\\ 12t \ln(1.0167) \amp = \ln(3)\\ t \amp = \frac{\ln(3)}{12 \ln(1.0167)}\\ t \amp \approx 5.53\text{.} \end{align*}
