We see that: 1. For \(\int\,x\sin x\,dx\text{,}\) integration by parts is appropriate, because by taking \(u = x\) and \(v' = \sin(x)\text{,}\) we end up with an integral that is easy to find. 2. For \(\int\,{x\over 1 + x^{2}}\,dx\text{,}\) substitution is appropriate, because by taking \(w=1 + x^{2}\) we get an integral we can find. 3. For \(\int\,x e^{x^{2}}\,dx\text{,}\) substitution is appropriate, because by taking \(w = x^{2}\) we get an integral we can find. 4. For \(\int\,x \cos(x^{2})\,dx\text{,}\) substitution is appropriate, because by taking \(w = x^{2}\) we get an integral we can find. 5. For \(\int\,{1\over\sqrt{5 x + 1}}\,dx\text{,}\) substitution is appropriate, because by taking \(w = 5 x + 1\) we get an integral we can find.
