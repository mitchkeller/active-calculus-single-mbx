Solution 5.4.7.8.1.

\(\int\,u\,v'\,dx = u\cdot v - \int u'\,v\,dx\)) with \(u = t^2\) and \(v' = e^{t}\text{,}\) so that \(u' = 2t\) and \(v = e^{t}\text{.}\) Then
\begin{equation*} \int\,4 t^2\,e^{t}\,dt = 4 t^2 e^{t} - \int 8 t e^{t}\,dt, \end{equation*}
and we can find the second integral by using integration by parts again. Here, we let \(u = t\) and \(v' = e^{t}\text{,}\) so that \(u' = 1\) and \(v = e^{t}\text{,}\) and
\begin{equation*} \int\,4 t^2\,e^{t}\,dt = 4 t^2 e^{t} - 8 t e^{t} + \int\,8 e^{t}\,dt \end{equation*}
\begin{equation*} = 4 t^2 e^{t} - 8 t e^{t} + 8 e^{t} + C. \end{equation*}
