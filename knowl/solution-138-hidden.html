The tangent function defined everywhere except where the cosine function is zero, so its domain is all real numbers except \(\pm \frac{\pi}{2}, \pm \frac{3\pi}{2}, \pm \frac{5\pi}{2}, \ldots\text{.}\) Using the quotient rule,
\begin{equation*} \begin{align*} f'(x) \amp = \frac{\cos(x)\cos(x)+\sin(x)\sin(x)}{\cos^2(x)} \\ \amp = \frac{1}{\cos^2(x)} \\ \amp = \sec^2(x). \end{align*} \end{equation*}
The derivative has the same domain as \(f\text{.}\)
