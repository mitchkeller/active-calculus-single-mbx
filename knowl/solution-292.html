Solution 5.1.5.3.1.

\(x=3\text{,}\) we are given that \(f(3)=0\text{.}\) Moving to the left on the interval \(2 \lt x \lt 3\text{,}\) we have \(f'(x) = -1\text{,}\) and we can find \(f(2)\) by knowing that the change in \(f(x)\) over this interval is equal to the area under \(f'(x)\) between \(x=2\) and \(x=3\text{.}\) Thus \(f(3) - f(2) = (-1)(1)\text{,}\) so that \(f(2) = f(3) - (1)(-1) = 1\text{.}\) Similarly, continuing back to \(x=0\text{,}\)
\begin{equation*} f(0) = f(2) - (1)(1) - (1)(-1) = 1 \end{equation*}
Moving to the right from \(x=3\text{,}\) we know that \(f'(x) = -1\) on \(3 \lt x \lt 4\text{.}\) So, as before, \(f(4) = f(3) + (1)(-1) = -1\text{.}\) Similarly, continuing to \(x=7\text{,}\)
\begin{equation*} f(7) = f(4) + (2)(1) + (1)(-1) = 0. \end{equation*}
Then, the value of the integral \(\int_0^7 f'(x)\,dx\) is given by the sum
\begin{equation*} \int_0^7 f'(x)\,dx = (-1)(1) + (1)(1) + (-1)(2) + (1)(2) + (-1)(1) = -1. \end{equation*}
Alternatively, knowing \(f(7)=0\) and \(f(0)=1\) and using the Fundamental Theorem of Calculus, we have
\begin{equation*} \int_0^7 f'(x)\,dx=f(7)-f(0) = (0) - (1) = -1. \end{equation*}
