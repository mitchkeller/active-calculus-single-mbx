\begin{equation*} \frac{4 x + 4}{x^2-3x+2} = \frac{4 x+4}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}. \end{equation*}
Multiplying by \((x-1)\) and \((x-2)\text{,}\) this becomes
\begin{equation*} 4 x+4 = A(x-2)+B(x-1), \end{equation*}
or
\begin{equation*} A+B = 4 \quad\hbox{and}\quad -2A - B = 4. \end{equation*}
Solving this system yields \(A = -8\) and \(B = 12\text{.}\) Then
\begin{equation*} \int\frac{4 x+4}{x^2-3x+2}dx = \int\left(\frac{12}{x-2} - \frac{8}{x-1}\right)\,dx = 12\ln\!\left(\left|x-2\right|\right)-8\ln\!\left(\left|x-1\right|\right)+C. \end{equation*}
