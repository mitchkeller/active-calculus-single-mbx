Solution 8.2.4.2.1.

\begin{equation*} \begin{array}{lcll} T(x) \amp = \amp \amp f(a) \cr\cr \amp \amp + \amp \frac{f'(a)}{1!} (x-a) \cr \cr \amp \amp + \amp \frac{f''(a)}{2!} (x-a)^2 \cr \cr \amp \amp + \amp \ldots \cr \cr \amp \amp + \amp \frac{f^n(a)}{n!} (x-a)^n \end{array} \end{equation*}
So, in this case, \(f(x) = \cos\!\left(x\right)\) and \(T_2(x) = A + B(x-\frac{\pi }{6}) + C(x-\frac{\pi }{6})^2\) where \(A=f(\frac{\pi }{6})=\frac{\sqrt{3}}{2}=0.866025403784439\) \(B=f'(\frac{\pi }{6})=-\frac{1}{2}=-0.5\) \(C=\frac{f''(\frac{\pi }{6})}{2!}=-\frac{\sqrt{3}}{4}=-0.433012701892219\) Thus,
\begin{equation*} T_2(x) = \frac{\sqrt{3}}{2} - \frac{1}{2} (x - \frac{\pi }{6}) - \frac{\sqrt{3}}{4} (x - \frac{\pi }{6})^2 \end{equation*}
or
\begin{equation*} T_2(x) = 0.866025-0.5\!\left(x-\frac{\pi }{6}\right)-0.433013\!\left(x-\frac{\pi }{6}\right)^{2} \end{equation*}
Similarly, \(T_3(x) = D + E(x-\frac{\pi }{6}) + F(x-\frac{\pi }{6})^2 + G(x-\frac{\pi }{6})^3\) where \(D=f(\frac{\pi }{6})=\frac{\sqrt{3}}{2} = 0.866025403784439\) \(E=f'(\frac{\pi }{6})=-\frac{1}{2} =-0.5\) \(F=\frac{f''(\frac{\pi }{6})}{2!}=-\frac{\sqrt{3}}{4} =-0.433012701892219\) \(G=\frac{f'''(\frac{\pi }{6})}{3!}=\frac{1}{12}=0.0833333333333333\) Thus,
\begin{equation*} T_3(x) = \frac{\sqrt{3}}{2} - \frac{1}{2} (x - \frac{\pi }{6}) - \frac{\sqrt{3}}{4} (x - \frac{\pi }{6})^2 + \frac{1}{12} (x - \frac{\pi }{6})^3 \end{equation*}
or
\begin{equation*} T_3(x) = 0.866025-0.5\!\left(x-\frac{\pi }{6}\right)-0.433013\!\left(x-\frac{\pi }{6}\right)^{2}+0.0833333\!\left(x-\frac{\pi }{6}\right)^{3} \end{equation*}
