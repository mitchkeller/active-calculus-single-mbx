(a) Suppose we choose an \(x\text{,}\) \(0\le x\le 20\text{.}\) We approximate the density of the number of the cars between \(x\) and \(x+\Delta x\) miles as \(\delta(x)\) cars per mile. Therefore, the number of cars between \(x\) and \(x+\Delta x\) is approximately \(\delta(x)\Delta x\text{.}\) If we slice the 20 mile strip into \(N\) slices, we get that the total number of cars is
\begin{equation*} C \approx \Sigma_{i=1}^N \delta(x_i)\Delta x = \Sigma_{i=1}^N \left[ 550 + 275\sin(4\sqrt{x_i + 0.125})\right]\Delta x, \end{equation*}
where \(\Delta x =20/N\text{.}\) This is a right-hand approximation; the corresponding left-hand approximation is \(\Sigma_{i=0}^{N-1}\delta(x_i)\Delta x\text{.}\) We can more loosely write both approximations without referring to the number of steps as
\begin{equation*} C \approx \Sigma \left[ 550 + 275\sin(4\sqrt{x + 0.125})\right]\Delta x. \end{equation*}
(b) As \(N\to\infty,\) the Riemann sum above approaches the integral
\begin{equation*} C =\int_0^{20} (550 + 275\sin4\sqrt{x + 0.125})\,dx. \end{equation*}
If we calculate the integral numerically, we find \(C\approx 10565.7\text{.}\) We can also find the integral exactly as follows:
\begin{equation*} C = \int_0^{20} (550 + 275\sin4\sqrt{x +0.125})\,dx = \int_0^{20} 550\,dx +\int_0^{20} 275\sin4\sqrt{x + 0.125}\,dx \end{equation*}
\begin{equation*} = 11000 + 275\int_0^{20}\sin4\sqrt{x +0.125}\,dx. \end{equation*}
Let \(w = \sqrt{x +0.125},\) so \(x =w^2 -0.125\) and \(dx =2w\,dw\text{.}\) Then
\begin{equation*} \int_{x=0}^{x=20}\sin 4\sqrt{x +0.125}\,dx = 2\int_{w=\sqrt{0.125}}^{w=\sqrt{20.125}} w\sin 4w\,dw, = 2\left[-{1\over 4}w\cos 4w +{1\over 16}\sin 4w\right] \bigg|_{\sqrt{0.125}}^{\sqrt{20.125}} \approx -1.46. \end{equation*}
(To find the integral we used the appropriate entry in a table of integrals.) Using this, we have \(C\approx 11000 + 275(-1.46) \approx 10565.7\text{,}\) which matches our numerical approximation.
