\begin{equation*} \frac{2x}{x^2-9}=\frac{A}{x+3}+\frac{B}{x-3} \end{equation*}
so that
\begin{equation*} 2x = A(x-3)+B(x+3) = (A+B)x + 3 (-A+B). \end{equation*}
Solving, we get \(A=B=1\text{,}\) so that
\begin{equation*} \int \frac{2x}{x^2-9}\,dx= \int \left(\frac{1}{x+3}+\frac{1}{x-3}\right)\,dx =\ln|x+3|+\ln|x-3|+C. \end{equation*}
Using the substitution \(w=x^2-9\text{,}\) we get \(dw= 2x\,dx\text{,}\) so we have
\begin{equation*} \int \frac{2x}{x^2-9}\,dx= \int \frac{dw}{w}=\ln|w|+C = \ln|x^2-9|+C. \end{equation*}
The properties of logarithms show that the two results are the same:
\begin{equation*} \ln|x+3|+\ln|x-3| = \ln|(x+3)(x-3)| =\ln|x^2-9|. \end{equation*}
