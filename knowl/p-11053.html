  3. We see that for every \(k\text{,}\) \(c_k = 1\text{.}\) Thus,
    \begin{equation*} T_5(x) = 1 + x + x^2 + \cdots + x^5 \end{equation*}
    and more generally
    \begin{equation*} T_n(x) = 1 + x + x^2 + \cdots + x^n\text{.} \end{equation*}
  4. Note that each subsequent term in \(T_n(x)\) results from multiplying the previous term by \(x\text{,}\) so \(T_n(x)\) is a finite geometric sum with \(r = x\) and \(a = 1\text{.}\)
  5. If we let the finite sum continue indefinitely, we see that the Taylor series centered at \(a = 0\) for \(f(x) = \frac{1}{1-x}\) is
    \begin{equation*} T(x) = 1 + x + x^2 + \cdots + x^n + \cdots \end{equation*}
    which is an infinite geometric sum with \(a = 1\) and \(r = x\) whose sum (for \(|r| = |x| \lt 1\)) is \(\frac{a}{1-r} = \frac{1}{1-x} = f(x)\text{.}\)
