So far, we have found the following approximations for \(f(x) = e^x\) near \(a = 0\text{:}\)
  • \(\displaystyle T_1(x) = 1 + x\)
  • \(\displaystyle T_2(x) = 1 + x + \frac{1}{2}x^2\)
  • \(\displaystyle T_3(x) = 1 + x + \frac{1}{2}x^2 + \frac{1}{6}x^3\)
  • \(\displaystyle T_4(x) = 1 + x + \frac{1}{2}x^2 + \frac{1}{6}x^3 + \frac{1}{24}x^4\)
In SectionÂ 8.2, we will develop a general approach to finding the coefficient of \(x^n\) in the degree \(n\) approximation; for now, we will use the \(4\) functions above and study further how the accuracy of the approximation changes with the degree.
