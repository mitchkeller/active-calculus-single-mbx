The Direct Comparison Test.

If

\begin{equation*} 0 \leq b_k \leq a_k \end{equation*}

for every \(k\text{,}\) then we must have

\begin{equation*} 0 \leq \sum b_k \leq \sum a_k \end{equation*}

  1. If \(\sum a_k\) converges, then \(\sum b_k\) converges.

  2. If \(\sum b_k\) diverges, then \(\sum a_k\) diverges.
