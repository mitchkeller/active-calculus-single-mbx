a. The function \(f\) is increasing where its derivative \(f'\) (as shown in the graph) is positive, or above the x-axis, so \(\left(0,1.5\right)\cup \left(4,6\right)\text{.}\) The function \(f\) is decreasing where its derivative \(f'\) is negative, or below the x-axis, so \(\left(1.5,4\right)\text{.}\)
b. The function \(f\) is concave up where its derivative \(f'\) is increasing, so \(\left(0,1\right)\cup \left(3,5\right)\text{.}\) The function \(f\) is concave down where its derivative \(f'\) is decreasing, so \(\left(1,3\right)\cup \left(5,6\right)\text{.}\)
c. By the First Derivative Test, the function \(f\) has a minimum where its derivative \(f'=0\) (so crosses the x-axes) and changed from negative to positive, so \(x=4\text{.}\) By the First Derivative Test, the function \(f\) has a maximum where its derivative \(f'=0\) (so crosses the x-axes) and changed from positive to negative, so \(x=1.5\text{.}\)
d. We use geometry to calculate the (signed) area to see that \(\int_0^1 f'(x) \, dx = \frac{1}{2}\text{,}\) and we were given that \(f(0)=1\text{,}\) so
\begin{equation*} f(1) - f(0) = \int_0^1 f'(x) \, dx \end{equation*}
means
\begin{equation*} f(1) - 1= \frac{1}{2}, \end{equation*}
so \(f(1) = 1.5\text{.}\)
Similarly,
\begin{equation*} f(2) - f(0) = \int_0^2 f'(x) \, dx, \end{equation*}
and using geometry, remembering that below the x-axis counts negatively, we see that \(\int_0^2 f'(x) \, dx = \frac{1}{2}\text{,}\) and so \(f(2) = 1.5\text{.}\) And continue in this way to calculate that \(f(3) = -0.5\text{,}\) \(f(4) = -2\text{,}\) \(f(5) = -0.5\text{,}\) \(f(6) = 1\text{.}\)
