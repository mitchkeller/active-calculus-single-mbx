SOLUTION a) Using the average rate of change formula above for the endpoints \((-1, f(-1))\) and \(( 4 , f( 4 ))\) , we obtain the following answer:
\begin{equation*} \frac{ f( 4 ) - f(-1) }{4 - (-1)} = \frac{ 4 + 11 }{4 + 1} = \frac{ 15}{5 } = 3 \end{equation*}
b) Using the average rate of change formula above for the endpoints \((a, f(a))\) and \(( b , f(b))\) , we obtain the following answer:
\begin{equation*} \frac{ f(b) - f(a) }{b - a} = \frac{ (3 b - 8) - (3 a - 8) }{b - a} = \frac{ 3 b - 3 a}{b-a} = \frac{ 3 (b-a) }{b-a} = 3 \end{equation*}
c) Using the average rate of change formula above for the endpoints \((x, f(x))\) and \(( x+h , f(x+h))\) , we obtain the following answer:
\begin{equation*} \frac{ f(x+h) - f(x) }{(x+h) - x} = \frac{ (3 (x+h) - 8) - (3 x - 8) }{h} = \frac{ 3 x + 3 h -8 - 3 x + 8}{h} = \frac{ 3 h }{h} = 3 \end{equation*}
