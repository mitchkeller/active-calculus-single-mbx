Solution 8.6.4.2.1.

\(\displaystyle c_n = \frac{f^{(n)}(0)}{n!}\text{,}\) e.g. \(\displaystyle c_0 = \frac{f(0)}{0!} = 0\text{.}\) \(\displaystyle c_1 = \frac{f'(0)}{1!}\text{;}\) \(\displaystyle f'(x)=8 \arctan(2 x) + 8 x\frac{1}{(2 x)^2+1}2 = 8 \arctan(2 x) + \frac{16 x}{4 x^2+1}\text{;}\) \(c_1 = 0\text{.}\) \(\displaystyle c_2 = \frac{f''(0)}{2!}\text{;}\) \(\displaystyle f''(x) = \left( 8 \arctan(2 x) + \frac{16 x}{4 x^2+1} \right)'\)\(\displaystyle = \frac{16}{4 x^2+1} + \frac{16(4 x^2+1)-16 x4\cdot 2x}{(4 x^2+1)^2}\)\(\displaystyle = \frac{64 x^2 + 16}{(4 x^2+1)^2} + \frac{16 - 64 x^2}{(4 x^2+1)^2} = \frac{32}{(4 x^2+1)^2}\text{;}\) \(\displaystyle c_2 = \frac{32}{2} = 16\text{;}\) and so on. Another way is to use the power series for arctan: \(\displaystyle \arctan(t) = \sum_{n=0}^{\infty} (-1)^n \frac{t^{2n+1}}{2n+1} = t-\frac{t^3}{3}+\frac{t^5}{5}-\frac{t^7}{7}+...\text{.}\) Then \(\displaystyle 8 x \arctan(2 x) = 8 x \sum_{n=0}^\infty (-1)^n \frac{(2 x)^{2n+1}}{2n+1} = \sum_{n=0}^\infty (-1)^n \frac{8\cdot 2^{2n+1}}{2n+1} x^{2n+2}\)\(\displaystyle = 16 x^2 - \frac{64}{3} x^4 + \frac{256}{5} x^6 - \frac{1024}{7} x^7+...\text{.}\) So \(c_0=0\text{,}\) \(c_1=0\text{,}\) \(c_2=16\text{,}\) \(c_3=0\text{,}\) \(c_4=-\frac{64}{3}=-21.3333333333333\text{...}\) This is an alternating series. It converges if and only if \(\displaystyle \lim_{n\to\infty}\frac{8 \cdot (2 x)^{2n+1}}{2n+1}=0\text{.}\) This limit is equal to 0 if and only if \(|2 x| \le 1\text{,}\) so \(\displaystyle |x|\le \frac{1}{2}\text{.}\) Therefore the radius of convergence is \(\displaystyle \frac{1}{2}\text{.}\)
