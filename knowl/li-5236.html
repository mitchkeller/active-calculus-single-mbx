Item 8.2.3.9.d.

For the third payment we are charged \(\left(\frac{r}{12}\right)P_2\) dollars in interest. So \(M-\left(\frac{r}{12}\right)P_2\) dollars goes toward paying down the principal. Then

\begin{equation*} P_3 = P_2 - \left( M - \left(\frac{r}{12}\right)P_2 \right) = \left(1 + \frac{r}{12}\right)P_2 - M\text{.} \end{equation*}

So \(P_3 = \left(1 + \frac{r}{12}\right)P_2 - M\text{,}\) so we substitute for \(P_2\) and simplify:

\begin{align*} P_3 \amp = \left(1 + \frac{r}{12}\right)P_2 - M\\ \amp = \left(1 + \frac{r}{12}\right)\left[\left(1 + \frac{r}{12}\right)^2P - \left[1 + \left(1+\frac{r}{12}\right)\right] M\right] - M\\ \amp = \left(1 + \frac{r}{12}\right)^3P - \left[1 + \left(1+\frac{r}{12}\right) + \left(1+\frac{r}{12}\right)^2 \right] M\text{.} \end{align*}
