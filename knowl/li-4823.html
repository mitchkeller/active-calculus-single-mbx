Item f.
We let \(q\) be the unknown intravenous rate of morphine (in milligrams per hour). Notice that if we replace \(3\) with \(q\) in the differential equation and obtain \(\frac{dM}{dt} = q - kM\text{,}\) the solution process is the same and the equilibriume solution is \(M = \frac{q}{k}\text{.}\) So if we want the equilibrium solution to be \(M = 7\text{,}\) we then have \(\frac{q}{k} = 7\text{,}\) so \(q = 7k \approx 2.426\text{.}\) The intravenous rate of morphine should be 2.426 milligrams per hour.
