\begin{equation*} \hbox{Volume of slab } = \pi r^2\Delta h = 36 \pi \Delta h \end{equation*}
\begin{equation*} \hbox{Distance to raise } = h \end{equation*}
\begin{equation*} \hbox{Work to raise } = (36 \pi)(50)(h) \Delta h. \end{equation*}
Integrating the work over all such slabs, we have
\begin{equation*} \hbox{Work} = \int_{16}^{30} (50)(36\pi)(h) \,dh = 900\pi h^2{\bigg|}_{16}^{30} = 579600 \pi \hbox{ lbf}. \end{equation*}
