Solution 1.2.4.8.1.

  1. \(f\) is defined for every value of \(x\) except those that make \(x^2 - 4 = 0\text{.}\) Hence, the domain of \(f\) is the set of all real numbers except \(x = \pm 2\text{.}\)
  2. Using a spreadsheet, we generate the following table.
    \(x\) \(f(x)\)
    \(2.1\) \(-8.41\)
    \(2.01\) \(-8.0401\)
    \(2.001\) \(-8.004001\)
    \(1.999\) \(-7.996001\)
    \(1.99\) \(-7.9601\)
    \(1.9\) \(-7.61\)
    The numerical trend suggests that \(\lim_{x \to 2} f(x) = -8\text{.}\)
  3. By factoring the numerator and denominator of \(f(x)\text{,}\) we observe that
    \begin{equation*} \frac{16-x^4}{x^2-4} = \frac{(4-x^2)(4+x^2)}{(x-2)(x+2)} = \frac{(2-x)(2+x)(4+x^2)}{(x-2)(x+2)}\text{.} \end{equation*}
    Taking the limit as \(x \to 2\text{,}\) we take values closer and closer, but not equal to 2, and therefore
    \begin{equation*} \lim_{x \to 2} \frac{16-x^4}{x^2-4} = \lim_{x \to 2} \frac{(2-x)(2+x)(4+x^2)}{(x-2)(x+2)} = \lim_{x \to 2} -1(4+x^2)\text{,} \end{equation*}
    since \(\frac{2-x}{x-2} = -1\) for all \(x \ne 2\text{.}\) Finally, evaluating the last limit as \(x \to 2\text{,}\) we see that
    \begin{equation*} \lim_{x \to 2} \frac{16-x^4}{x^2-4} = -1(4+2^2) = -8\text{.} \end{equation*}
