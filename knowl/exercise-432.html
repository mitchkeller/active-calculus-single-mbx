Exercise 3.

What is the correct form of the partial fraction decomposition for the following integral?
\begin{equation*} \int \frac{x - 10}{x^3 + 6 x^2 - 27 x} \, dx \end{equation*}
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x} + \frac{B}{x - 3} + \frac{C}{x + 9} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A x + B}{x} + \frac{C x + D}{x - 3} + \frac{E x + F}{x + 9} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x} + \frac{B}{x - 10} + \frac{C}{x + 3} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 10} + \frac{B}{x - 9} + \frac{C}{x + 3} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x} + \frac{B x + C}{x - 10} + \frac{D x + E}{x + 3} \right)\, dx\)
  • There is no partial fraction decomposition because the denominator does not factor.
  • There is no partial fraction decomposition yet because there is cancellation.
  • There is no partial fraction decomposition yet because long division must be done first.
