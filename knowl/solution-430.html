Solution 7.1.5.6.1.

\(t=0\) the value of \(y\) is 6. Since we are told that \(dy/dt=0.8 y\text{,}\) we know that at time \(t=0\) the derivative of \(y\) is \(0.8 (6) = 4.8\text{.}\) Thus as \(t\) goes from 0 to 1, \(y\) will increase by 4.8, so at \(t=1\text{,}\) \(y=6 + 4.8 = 10.8\text{.}\) Likewise, at \(t=1\text{,}\) we get \({dy}/{dt} = 0.8 (10.8)=8.64\) so that at \(t=2\text{,}\) we obtain \(y=10.8 + 8.64 = 19.44\text{.}\) At \(t=2\text{,}\) we have \(dy/dt=0.8 (19.44)=15.552\) so that at \(t=3\text{,}\) we obtain \(y=19.44 + 15.552 = 34.992\text{.}\) Continuing similarly, \(dy/dt|_{t=3} = 27.9936\text{,}\) so that at \(t=4\) we have \(y = 34.992 + 27.9936 = 62.9856\text{.}\)
