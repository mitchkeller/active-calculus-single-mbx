We find inflection points by determining where \(f''(x)\) changes sign. \(f'(x) = 4\cdot 4x^{3}+55\cdot 3x^{2}-21\cdot 2x\text{,}\) so \(f''(x) = 4\cdot 4\cdot 3x^{2}+55\cdot 3\cdot 2x-42 = 6(8 x - 1)(x + 7)\text{.}\) So we see that \(f''(x)\) changes sign at \(x = 1/8\) and \(x = -7\text{.}\)
