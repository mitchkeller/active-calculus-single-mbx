Exercise 3.

What is the correct form of the partial fraction decomposition for the following integral?
\begin{equation*} \int \frac{x - 10}{x^3 + 4 x^2 - 45 x} \, dx \end{equation*}
  • A. \(\displaystyle \int \left( \frac{A}{x} + \frac{B}{x - 5} + \frac{C}{x + 9} \right)\, dx \)
  • B. \(\displaystyle \int \left( \frac{A}{x} + \frac{B x + C}{x - 10} + \frac{D x + E}{x + 5} \right)\, dx \)
  • C. There is no partial fraction decomposition because the denominator does not factor.
  • D. \(\displaystyle \int \left( \frac{A}{x - 10} + \frac{B}{x - 9} + \frac{C}{x + 5} \right)\, dx \)
  • E. There is no partial fraction decomposition yet because there is cancellation.
  • F. There is no partial fraction decomposition yet because long division must be done first.
  • G. \(\displaystyle \int \left( \frac{A}{x} + \frac{B}{x - 10} + \frac{C}{x + 5} \right)\, dx \)
  • H. \(\displaystyle \int \left( \frac{A x + B}{x} + \frac{C x + D}{x - 5} + \frac{E x + F}{x + 9} \right)\, dx \)
