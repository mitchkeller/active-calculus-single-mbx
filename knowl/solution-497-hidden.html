\(x\) = a:
\begin{equation*} T(x) = f(a) + \frac{f'(a)}{1!} (x-a) + \frac{f''(a)}{2!} (x-a)^2 + ... + \frac{f^n(a)}{n!} (x-a)^n \end{equation*}
\begin{equation*} f(x) = \ln\!\left(x+1\right) \end{equation*}
\begin{equation*} f'(x) = \frac{1}{x+1} \end{equation*}
\begin{equation*} f''(x) = \frac{-1}{\left(x+1\right)^{2}} \end{equation*}
\begin{equation*} f'''(x) = \frac{2}{\left(x+1\right)^{3}} \end{equation*}
So, in this case, \(f(x) = \ln\!\left(x+1\right)\) and \(T_2(x) = A + B(x-1) + C(x-1)^2\) where \(A=f(1)=\ln (2)=0.693147\) \(B=f'(1)=\frac{1}{(2)}=0.5\) \(C=\frac{f''(1)}{2!}=-\frac{1}{(8)}=-0.125\) Thus,
\begin{equation*} T_2(x) = \ln (2) + \frac{1}{(2)} (x - 1) - \frac{1}{(8)} (x - 1)^2 \end{equation*}
or,
\begin{equation*} T_2(x) = 0.693147+0.5\!\left(x-1\right)-0.125\!\left(x-1\right)^{2} \end{equation*}
Similarly, \(T_3(x) = D + E(x-1) + F(x-1)^2 + G(x-1)^3\) where \(D=f(1)=\ln (2)=0.693147\) \(E=f'(1)=\frac{1}{(2)}=0.5\) \(F=\frac{f''(1)}{2!}=-\frac{1}{(8)}=-0.125\) \(G=\frac{f'''(1)}{3!}=\frac{1}{(24)}=0.0416667\) Thus,
\begin{equation*} T_3(x) = \ln (2) + \frac{1}{(2)} (x - 1) - \frac{1}{(8)} (x - 1)^2 + \frac{1}{(24)} (x - 1)^3 \end{equation*}
or,
\begin{equation*} T_3(x) = T_2(x) + 0.0417 (x-1)^3 \end{equation*}
