\(k\text{,}\) the function is continuous on any interval that does not contain \(x=2\text{.}\) Since \(4 x^{3} - 8 x^{2} = 4 x^{2}(x - 2)\text{,}\) we can cancel \((x-2)\) provided \(x\ne 2\text{,}\) giving
\begin{equation*} f(x) = \frac{4x^{3}-8x^{2}}{x-2} = 4 x^{2},\quad\quad x\ne 2. \end{equation*}
Thus, if we pick \(k = 4 (2)^{2} = 16\text{,}\) the function is continuous.
