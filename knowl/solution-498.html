Solution 8.2.4.5.1.

\(f(0) = 7\text{;}\) \(f'(x) = 4 x - 5\text{,}\) so that \(f'(0) = -5\text{;}\) and \(f''(x) = 4\text{,}\) so that \(f''(0) = 4\text{.}\) Thus
\begin{equation*} T_2(x) = 7 - 5 x + {4\over 2!} x^2 = 7 - 5 x + 2 x^2. \end{equation*}
We notice that \(f(x) = T_2(x)\) in this case, which makes sense because \(f(x)\) is a polynomial.
