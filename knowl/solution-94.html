Solution 1.8.4.1.1.

\(f(130) = 85\) and \(f'(130) = 3\) tell us that at \(x = 130\text{,}\) the value of the function is 85 and the function is increasing at a rate of 3 units for a unit increase in \(x\text{.}\) Since we increase \(x\) by 0.5 units in going from 130 to 130.5, the value of the function goes up by approximately \(0.5 \cdot 3 = 1.5\) units, so
\begin{equation*} f(130.5) \approx 85 + 0.5 \cdot 3 = 85 + 1.5 = 86.5. \end{equation*}
