Exercise 4.

What is the correct form of the partial fraction decomposition for the following integral?
\begin{equation*} \int \frac{3 (x^{8} + 7)}{(x - 8)(x^2 - 1)^2 (x^2 + 8)^2} \, dx \end{equation*}
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 8} + \frac{Bx+C}{(x^2-1)^2} + \frac{Dx+E}{(x^2 + 8)^2} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 8} + \frac{B}{x+1} + \frac{C}{(x+1)^2} + \frac{D}{x-1} + \frac{E}{(x-1)^2} + \frac{Fx+G}{x^2 + 8} \right)\, dx\)
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 8} + \frac{B}{(x^2-1)^2} + \frac{C}{(x^2 + 8)^2} \right)\, dx\)
  • There is no partial fraction decomposition because the denominator does not factor.
  • There is no partial fraction decomposition yet because there is cancellation.
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 8} + \frac{Bx+C}{x^2-1} + \frac{Cx+D}{(x^2-1)^2} + \frac{Ex+F}{x^2 + 8} + \frac{Gx+H}{(x^2 + 8)^2} \right)\, dx\)
  • There is no partial fraction decomposition yet because long division must be done first.
  • \(\displaystyle \displaystyle \int \left( \frac{A}{x - 8} + \frac{B}{x+1} + \frac{C}{(x+1)^2} + \frac{D}{x-1} + \frac{E}{(x-1)^2} + \frac{Fx+G}{x^2 + 8} + \frac{Hx+I}{(x^2 + 8)^2} \right)\, dx\)
