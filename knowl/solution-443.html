Solution 1.

\(y\text{,}\) we can compute the slope of the tangent line using the formula for \(dy/dt\text{.}\) So given a value of \(t\text{,}\) we must first compute (or approximate) a value of \(y\text{.}\) When \(t=0\text{,}\) the initial condition tells us that \(y=0\text{,}\) so the corresponding slope is \(\frac{1}{2}(0+1) = \frac{1}{2}\) and the equation of the tangent line is \(y=\frac{1}{2}t\text{.}\) Now when \(t=2\text{,}\) we can use this tangent line equation to approximate \(y \approx \frac{1}{2}\cdot 2 =1\text{,}\) and then we can use this approximate value of \(y\) to approximate the slope to be \(\frac{1}{2}(1+1)\text{.}\) In point-slope form, the tangent line to the solution curve at \(t=2\) is approximately \(y\approx 1(t-2)+1\text{,}\) and so \(y(4) \approx 3\text{.}\)
