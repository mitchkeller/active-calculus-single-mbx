Item 3.3.4.8.d.
Finally, for \(s(x) = \sin(kx)\) on \(\left[\frac{\pi}{3k}, \frac{5\pi}{6k}\right]\text{,}\) we know \(s'(x) = k\cos(kx)\text{,}\) and \(\cos(kx) = 0\) when \(kx = \frac{\pi}{2} \pm n\pi\) for some integer \(n\text{,}\) so \(x = \frac{\pi}{2k} \pm n\pi = \frac{\pi \pm 2n\pi}{2k}\text{.}\) These \(x\)-values are the critical numbers of \(s\text{.}\) Note that when \(n = -1, 0, 1\text{,}\) the corresponding critical numbers are
\begin{equation*} x = -\frac{\pi}{2k}, \frac{\pi}{2k}, \frac{3\pi}{2k} \end{equation*}
Comparing the values of \(-\frac{\pi}{2}\text{,}\) \(\frac{\pi}{2k}\text{,}\) and \(\frac{3\pi}{2k}\) to \(\frac{\pi}{3}\) and \(\frac{5\pi}{6}\text{,}\) we see that only \(\frac{\pi}{2k}\) lies in the given interval \(\left[\frac{\pi}{3k}, \frac{5\pi}{6k}\right]\text{,}\) and thus \(x = \frac{\pi}{2k}\) is the only critical number at which we need to evaluate \(s\text{.}\)
Evaluating \(s\) at the endpoints and relevant critical number, we find:
  • \(s\left( \frac{\pi}{3k} \right) = \sin\left( k \cdot \frac{\pi}{3k} \right) = \sin\left( \frac{\pi}{3} \right) = \frac{\sqrt{3}}{2}\text{;}\)
  • \(s\left( \frac{5\pi}{6k} \right) = \sin\left( k \cdot \frac{5\pi}{6k} \right) = \sin\left( \frac{5\pi}{6} \right) = \frac{1}{2}\text{;}\) and
  • \(s\left( \frac{\pi}{2k} \right) = \sin\left( k \cdot \frac{\pi}{2k} \right) = \sin\left( \frac{\pi}{2} \right) = 1\text{.}\)
Hence the absolute max of \(s\) on \(\left[\frac{\pi}{3k}, \frac{5\pi}{6k}\right]\) is \(s\left( \frac{\pi}{2k} \right) = 1\) and the absolute min of \(s\) is \(s\left( \frac{5\pi}{6k} \right) = \frac{1}{2}\text{.}\)
