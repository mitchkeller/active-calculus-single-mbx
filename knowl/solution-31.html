Solution 1.3.3.4.1.

\begin{equation*} g'(7) = \lim_{h\to 0} \frac{g(7+h)-g(7)}{h}, \end{equation*}
so
\begin{equation*} g'(7) = \lim_{h\to 0}\frac{(2(7+h)^2 + 2(7+h)) - (2(7)^2 + 2(7))}{h}. \end{equation*}
Expanding the quadratic,
\begin{equation*} g'(7) = \lim_{h\to 0} \frac{(98 + 28 h + 2 h^2 + 14 + 2 h) - (98 + 14)} {h} \end{equation*}
\begin{equation*} = \lim_{h\to 0} \frac{ 28 h + 2 h^2 + 2 h }{h} = \lim_{h\to 0} 28 + 2 + 2 h \end{equation*}
\begin{equation*} = 28 + 2 = 30. \end{equation*}
