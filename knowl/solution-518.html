Solution 8.1.3.1.

Let \(f(x)=e^x\) and \(T_3(x) = k_0 + k_1 x + k_2 x^2 + k_3 x^3\text{.}\)
  2. Since \(T_3(0) = f(0)\text{,}\) this means \(k_0 = 1\text{.}\) Next, \(T_3'(0) = f'(0)\) tells us that \(k_1 = 1\text{.}\) Using \(T_3''(0) = f''(0)\text{,}\) \(2k_2 = 1\) so that \(k_2 = \frac{1}{2}\text{.}\) And finally, the last condition is \(T_3'''(0) = f'''(0)\text{,}\) which implies that \(6k_3 = 1\text{,}\) so \(k_3 = \frac{1}{6}\text{.}\)
  3. We notice that \(T_3(x)\) appears to be an even better approximation than \(T_2(x)\) near \(a = 0\) and that the quality of the approximation extends further. We can see the better approximation by \(T_3(x)\) in the figure, and if we do some computations, we can see that for certain \(x\) near \(a = 0\text{,}\) \(T_3(x)\) is closer to \(e^x\) than \(T_2(x)\) and that the set of \(x\)-values for which \(|f(x)-T_3(x)| \lt 0.1\) is wider than for which \(|f(x)-T_2(x)| \lt 0.1\text{.}\) We found in ActivityÂ 8.1.2 that \(|f(x)-T_2(x)| \lt 0.1\) for approximately \(-0.9 \lt x \lt 0.8\text{;}\) here, \(|f(x)-T_2(x)| \lt 0.1\) for approximately \(-1.3 \lt x \lt 1.1\text{.}\)
  4. \(T_4(x)\) involves the term \(w_4 x^4\text{.}\) When we differentiate this function \(4\) times, we get \(4 \cdot 3 \cdot 2 \cdot 1 w_4\text{.}\) Since \(f^{(4)}(x) = e^x\text{,}\) \(f^{(4)}(0) = 1\text{,}\) so \(4 \cdot 3 \cdot 2 \cdot 1 w_4 = 1\) and therefore \(w_4 = \frac{1}{24}\text{.}\) This means that
    \begin{equation*} T_4(x) = 1 + x + \frac{1}{2}x^2 + \frac{1}{6} x^3 + \frac{1}{24}x^4\text{.} \end{equation*}
    Plotting \(T_4(x)\) along with the earlier approximations we see that \(T_4(x)\) is an even better approximation to \(f(x) = e^x\) and on a still wider interval.
