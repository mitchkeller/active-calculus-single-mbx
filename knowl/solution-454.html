Solution 1.

\(g(y) \frac{dy}{dt} = h(t)\text{.}\) For example, \((1) \frac{dy}{dt} = t+1\text{,}\) \(\left( \frac{1}{y} \right) \frac{dy}{dt} = t-1\text{,}\) and \(\left( \frac{1}{y} \right) \frac{dy}{dt} = -3\) are all separable. The equation \(\frac{dy}{dt} = t^2 - y^2\) is not separable, however, because the right-hand side cannot be factored into a product of a function of \(y\) and a function of \(t\text{.}\) For the second part, note that a solution must be an antiderivative of \(x\text{.}\)
