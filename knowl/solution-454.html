Solution 6.5.5.10.1.

Substitute \(x = r \tan{\theta}\text{,}\) so that
\begin{equation*} u = Ar \displaystyle\int_{a}^{\infty} \frac{\hbox{d}x}{(r^2+x^2)^{(3/2)}} = Ar \displaystyle\int_{a}^{\infty} \frac{r\sec^2{\theta} \hbox{d}\theta}{(r^2(1+\tan^2{\theta}))^{(3/2)}} \end{equation*}
Then, since
\begin{equation*} \displaystyle\int \frac{r\sec^2{\theta} \hbox{d}\theta}{(r^2(1+\tan^2{\theta}))^{(3/2)}} = \displaystyle\int \frac{r\sec^2{\theta} \hbox{d}\theta}{(r^2(1+\tan^2{\theta}))(r^2(1 + \tan^2{\theta}))^{(1/2)}} \end{equation*}
\begin{equation*} = \displaystyle\int \frac{\hbox{d}\theta}{r^2\sec{\theta}} = \frac{1}{r^2} \sin{\theta} \end{equation*}
using a trig identity.
Then since \(x = r \tan{\theta} = r \displaystyle\frac{\sin{\theta}}{\cos{\theta}}\) we see
\begin{equation*} \sin{\theta} = \displaystyle\frac{x}{r}\cos{\theta} = \displaystyle\frac{x}{r\sec{\theta}} = \displaystyle\frac{x}{r\sqrt{1+ \tan^2{\theta}}} = \frac{x}{\sqrt{r^2 + x^2}} \end{equation*}
Finally,
\begin{equation*} u = \lim_{b\to\infty} \frac{A}{r}\left[\frac{x}{\sqrt{r^2+x^2}}\right]_{a}^{b} = \frac{A}{r}\left(1-\frac{a}{\sqrt{r^2+a^2}}\right) \end{equation*}
