Solution 2.6.6.1.1.

\((1,5)\) is on the curve, we know \(f(1)=5\text{.}\) The point \((1.1,5.4)\) is on the tangent line, so
\begin{equation*} \hbox{Slope of tangent} = \frac{5.4-5}{1.1-1} = \frac{0.4}{0.1} = 4. \end{equation*}
Thus, \(f'(1)=4\text{.}\) By the chain rule
\begin{equation*} h'(1) = 2(f(1))\cdot f'(1) =2\cdot 5\cdot 4 = 40. \end{equation*}
Because \(g(x)\) is the inverse function of \(f\text{,}\) we know that \(g(5) = 1\text{.}\) Then note that \(f(g(x)) = f(f^{-1}(x)) = x\text{,}\) so \(f(g(x)) = x\text{.}\) Differentiating both sides of the equation, \(f'(g(x))\cdot g'(x) = 1\text{,}\) so \(g'(x) = \frac{1}{f'(g(x))}\text{.}\) Thus \(g'(5) = \frac{1}{f'(g(5))} = \frac{1}{f'(1)} = \frac1{4}\text{.}\)
