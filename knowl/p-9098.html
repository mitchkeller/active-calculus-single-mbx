Since \(T_3(0) = f(0)\text{,}\) this means \(k_0 = 1\text{.}\) Next, \(T_3'(0) = f'(0)\) tells us that \(k_1 = 1\text{.}\) Using \(T_3''(0) = f''(0)\text{,}\) \(2k_2 = 1\) so that \(k_2 = \frac{1}{2}\text{.}\) And finally, the last condition is \(T_3'''(0) = f'''(0)\text{,}\) which implies that \(6k_3 = 1\text{,}\) so \(k_3 = \frac{1}{6}\text{.}\)
