\(\int\,u\,v'\,dx = u\cdot v - \int u'\,v\,dx\)) with \(u = z+1\) and \(v' = e^{8 z}\text{,}\) so that \(u' = 1\) and \(v = {1\over 8} e^{8 z}\text{.}\) Then
\begin{equation*} \int\,\left(z+1\right)e^{8z}\,dz = \frac{z+1}{8}\,e^{8 z} - \int\,{1\over 8} e^{8 z}\,dz = \frac{z+1}{8}\,e^{8 z} - {1\over 64} e^{8 z} + C. \end{equation*}
