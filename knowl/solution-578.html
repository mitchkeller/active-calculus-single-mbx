Solution 9.2.9.1.

\(Ax^n\) the antiderivative is just the term whose derivative is \(Ax^n\text{.}\) That is, \(\frac{A}{n+1}x^{n+1} + C\) where \(C\) is an unknown constant. Also, recall that the antiderivative of the \(\sec^2\) function is the \(\tan\) function. Applying these rules term-by-term to our function, we get:
\begin{equation*} F(t) = 8\tan{t} - \frac{7}{3}t^{3} + C \end{equation*}
\begin{equation*} \quad = 8\tan{t} - 2.33333333333333t^{3} + C \end{equation*}
Where \(C\) is an artibrary constant. We are further told that \(F(0) = 0\) so plugging this into the above general antiderivative, we get:
\begin{equation*} F(0) = 8\tan{0} - 2.33333333333333(0)^{3} + C = 0 \end{equation*}
So that the specific antiderivative is:
\begin{equation*} F(t) = 8\tan{t} - 2.33333333333333t^{3} \end{equation*}
