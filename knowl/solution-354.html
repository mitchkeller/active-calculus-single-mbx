Solution 5.3.5.8.1.

A. \(\int\,6\,x\,(x^{2} + 1)\,dx = \int\,6\,x^{3} + 6\,x\,dx\text{.}\) Thus, antidifferentiating term-by-term, we get \(\int\,6\,x\,(x^{2} + 1)\,dx = {3\over2}\,x^{4} + 3\,x^{2} + C\text{.}\) B. \(\int\,6\,x\,(x^{2} + 1)\,dx = \int\,3\,w\,dw\text{.}\) Thus, antidifferentiating, we get \(\int\,6\,x\,(x^{2} + 1)\,dx = {3\over2}\,w^2 + C = {3\over2}\,(x^{2} + 1)^2 + C\text{.}\) C. To see the difference between these two answers, we expand the second: \((x^{2} + 1)^2 = x^{4} + 2\,x^{2} + 1\text{,}\) so the difference is
\begin{equation*} ({3\over2} (x^{4} + 2\,x^{2} + 1)) - ({3\over2} \,x^{4} + 3\,x^{2}) = {3\over2}. \end{equation*}
That is, the difference is a constant! This is completely reasonable: the antiderivative is only determined up to a constant, and the arbitrary constant \(C\) that we get for the most general antiderivative will adjust for this difference.
