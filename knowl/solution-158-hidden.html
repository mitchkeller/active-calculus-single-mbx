\(H(x) = F(G(x))\text{,}\) \(H(1) = F(G(1)) = F(3) = 3\) \(H'(1) = F'(G(1))\cdot G'(1) = F'(3)\cdot 3 = 2\cdot 3 = 6\) For \(H(x) = G(F(x))\text{,}\) \(H(1) = G(F(1)) = G(2) = 6\) \(H'(1) = G'(F(1))\cdot F'(1) = G'(2)\cdot 7 = 1\cdot 7 = 7\) For \(H(x) = F(x)/G(x)\text{,}\) \(H'(1) = \frac{F'(1) G(1) - F(1) G'(1)}{G(1)^2} = \frac{ (7)(3) - (2)(3) } {3^2} \approx 1.667\text{.}\)
