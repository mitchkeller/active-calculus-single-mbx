We differentiate implicitly both sides of the equation with respect to \(x\text{.}\)

\begin{equation*} \ln y + \frac xy \, \frac{dy}{dx} + 3 y^{2}\frac{dy}{dx} = \frac{3}{x}, \end{equation*}

so

\begin{equation*} \left( \frac{x}{y} + 3 y^{2} \right) \, \frac{dy}{dx} = \frac{3}{x} - \ln y, \end{equation*}

and thus

\begin{equation*} \frac{dy}{dx} = \frac{\frac{3}{x} - \ln y}{\frac{x}{y} + 3 y^{2}} = \frac{y\!\left(3-x\ln\!\left(y\right)\right)}{x\!\left(x+3y^{3}\right)}. \end{equation*}
