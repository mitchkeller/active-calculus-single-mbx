Item 8.4.4.8.b.
With \(f(x) = \frac{1}{1+x} = (1+x)^{-1}\text{,}\) we observe that \(f'(x) = -(1+x)^{-2}\text{,}\) \(f''(x) = (-1)(-2)(1+x)^{-3}\text{,}\) \(f'''(x) = (-1)(-2)(-3)(1+x)^{-4}\text{,}\) and \(f^{(4)}(x) = (-1)(-2)(-3)(-4)(1+x)^{-5}\text{.}\) Evaluating each of these functions at \(a = 1\text{,}\) we find \(f(1) = \frac{1}{2}\text{,}\) \(f'(1) = (-1) \cdot \frac{1}{2^2}\text{,}\) \(f''(1) = \frac{2!}{2^3}\text{,}\) \(f'''(1) = (-1) \cdot \frac{3!}{2^4}\text{,}\) and \(f^{(4)}(1) = \frac{4!}{2^5}\text{.}\)
From these derivative values and the fact that \(c_k = \frac{f^{(k)}(1)}{k!}\text{,}\) it follows that
\begin{equation*} T_4(x) = \frac{1}{2} - \frac{1}{2^2}(x-1) + \frac{1}{2^3}(x-1)^2 - \frac{1}{2^4}(x-1)^3 + \frac{1}{2^5}(x-1)^4\text{,} \end{equation*}
from which we can also easily find \(T_3(x)\text{,}\) \(T_2(x)\text{,}\) and \(T_1(x)\text{.}\)
Finally, from the patterns in our preceding work, we see the Taylor series for \(f(x)\) centered at \(a = 1\) is
\begin{equation*} T(x) = \frac{1}{2} - \frac{1}{2^2}(x-1) + \frac{1}{2^3}(x-1)^2 - \frac{1}{2^4}(x-1)^3 + \frac{1}{2^5}(x-1)^4 + \cdots + \frac{1}{2^{n+1}}(x-1)^{n} + \cdots\text{.} \end{equation*}
