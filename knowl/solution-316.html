Solution 5.3.5.6.1.

\(w = \sin\!\left(q\right)\text{,}\) so that \(dw = \cos\!\left(q\right)\,dq\text{.}\) Then \(w(\pi/2) = 1\) and \(w(\pi) = 0\text{,}\) so that
\begin{equation*} \int_{\pi/2}^{\pi}\, e^{\sin\!\left(q\right)}\cdot \cos\!\left(q\right) \,dq = \int_1^0\,e^w\,dw = e^w\bigg|_1^0 = 1 - e. \end{equation*}
