Solution 7.4.3.3.1.

\begin{equation*} \frac{dy}{y^2} = (2 + t)\,dt, \end{equation*}
so
\begin{equation*} \int \frac{dy}{y^2} = \int (2 + t)\,dt, \end{equation*}
and
\begin{equation*} -\frac{1}{y} = {1\over2}(2 + t)^2 + C, \end{equation*}
giving
\begin{equation*} y=-\frac{2}{(2 + t)^2 + A}, \end{equation*}
where \(A = 2C\text{.}\) Since \(y=5\) when \(t=1\text{,}\) we have
\begin{equation*} 5 = -\frac{2}{3^2 + A},\quad \hbox{ so } \quad 5 A + 45 = -2, \end{equation*}
and \(A = -{{47\over5}}\text{.}\) Thus
\begin{equation*} y = -\frac{2}{(2 + t)^2 - {{47\over5}}}. \end{equation*}
