You have a rectangle of size 12 by 17, and you cut a square out of each corner. Say each square has length \(x\text{.}\) Then when you fold the rectangle up, the height of the resulting box is \(x\) and the length is \(17-2 x\) while the width is \(12 -2 x\text{.}\) So the volume of the box is
\begin{equation*} V=l \cdot w \cdot h = (17 -2 x) (12 -2 x)x= (204-58 x +4 x^2)x=204 x-58 x^2+4x^3 \end{equation*}
Taking the derivative and setting it equal to 0 gives
\begin{equation*} V'=204-2\cdot 58 x+ 12x^2 =0 \end{equation*}
Using the quadratic formula, we get that
\begin{equation*} x=\frac{2\cdot58\pm\sqrt{4\cdot58^2-4\cdot 12\cdot 204}}{2\cdot 12}= \end{equation*}
\begin{equation*} 2.31121 \mbox{ or } 7.35545765840359 \end{equation*}
But if \(x=7.35545765840359\text{,}\) then the width of the box, which is \(12-2 x\text{,}\) is negative. So the height is \(x=2.31121\text{.}\)
