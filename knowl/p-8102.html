We want to find the time it takes for \(M(t)\) to be \(30\)% of \(M_0\text{,}\) so we need to solve \(0.3M_0 = M_0e^{-\frac{\ln(2)}{5730}t}\text{.}\) Doing so in the usual way, we find that \(t = -\frac{5730\ln(0.3)}{\ln(2)} \approx 9952.8\) years, so we estimate that the sample is about \(9973\) years old.
