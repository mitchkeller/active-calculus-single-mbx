Solution 5.3.5.7.1.

\(w = -\cos\mathopen{}\left(q\right)\text{,}\) so that \(dw = \sin\mathopen{}\left(q\right)\,dq\text{.}\) Then \(w(3\pi) = 1\) and \(w(3\pi/2) = 0\text{,}\) so that
\begin{equation*} \int_{3\pi}^{3\pi/2}\, e^{-\cos\mathopen{}\left(q\right)}\cdot \sin\mathopen{}\left(q\right) \,dq = \int_1^0\,e^w\,dw = e^w\bigg|_1^0 = 1 - e. \end{equation*}
