Recall that the length of a curve defined by a function \(f\) on an interval \([a,b]\) is given by
\begin{equation*} \int_a^b \sqrt{1+(f'(x))^2} \ dx\text{.} \end{equation*}
To find the perimeter of the region \(R\) we calculate the length of the curves defined by \(y = 1+\frac{1}{2}(x-2)^2\) (for which \(y' = x-2\)) and \(y=\frac{1}{2}x^2\) (for which \(y' = x\)) on the interval \([0,1.5]\) and the length of the segment along \(x=0\) as \(y\) goes from \(0\) to \(3\text{.}\) This last length is \(3\text{,}\) so the perimeter of the region \(R\) is given by
\begin{equation*} P = 3 + \int_0^{1.5} \sqrt{1+(x-2)^2} + \sqrt{1+x^2} \ dx \approx 7.387234642\text{.} \end{equation*}
