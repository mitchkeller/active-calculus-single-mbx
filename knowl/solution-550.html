Solution 8.3.5.7.1.

  1. We consider the finite geometric series that results when \(a = 1\) and \(r = 1\text{:}\)
    \begin{equation*} S_n = 1 + 1 + \cdots + 1\text{,} \end{equation*}
    which has \(n\) terms in the sum.
    1. Using
      \begin{equation*} S_n = 1 + 1 + \cdots + 1\text{,} \end{equation*}
      we see that \(S_2 = 1 + 1 = 2\text{,}\) \(S_3 = 1 + 1 + 1 = 3\text{,}\) \(S_4 = 4\text{,}\) and more generally, \(S_n = n\text{.}\)
    2. Because the sequence \(1, 2, 3, 4, \ldots, n, \ldots\) increases without bound, partial sums of the series never approach a single finite value, so the sequence diverges, which means the infinite sum \(S = 1 + 1 + \cdots + 1 + \cdots\) also diverges.
  2. Next, we consider the finite geometric series that results when \(a = 1\) and \(r = -1\text{:}\)
    \begin{equation*} S_n = 1 - 1 + 1 - 1 + \cdots + (-1)^{n-1}\text{,} \end{equation*}
    which has \(n\) terms in the sum.
    1. Using \(S_n = 1 - 1 + 1 - 1 + \cdots + (-1)^{n-1}\text{,}\) we see that \(S_2 = 1 - 1 = 0\text{,}\) \(S_3 = 1 - 1 + 1 = 1\text{,}\) \(S_4 = 1 - 1 + 1 - 1 = 0\text{,}\) and \(S_5 = 1 - 1 + 1 - 1 + 1 = 1\text{,}\) and thus that \(S_n = 0\) when \(n\) is even, and \(S_n = 1\) when \(n\) is odd.
    2. Because the sequence \(1, 0, 1, 0, 1, 0, \ldots, 1, 0, \ldots\) alterates between the values \(0\) and \(1\) partial sums of the series never approach a single finite value, so the sequence diverges, which means the infinite sum \(S = 1 - 1 + 1 - 1 + \cdots + 1 - 1 + \cdots\) also diverges.
  3. Finally, we consider the geometric sum with \(a = 1\) and \(r = 2\text{:}\)
    \begin{equation*} S_n = 1 + 2 + 4 + \cdots + 2^{n-1}\text{.} \end{equation*}
    1. Using the formula \(S_n = \frac{a - ar^n}{1-4}\text{,}\) we see that \(S_2 = \frac{1-2^2}{-1} = 3\text{,}\) \(S_3 = \frac{1-2^3}{-1} = 7\text{,}\) and \(S_4 = \frac{1-2^4}{-1} = 15\text{,}\) so \(S_n = 2^n - 1\text{.}\)
    2. The partial sums \(S_n = 2^n - 1\) increase without bound, so the infinite geometric series \(1 + 2 + 4 + \cdots + 2^{n-1} + \cdots\) diverges.
