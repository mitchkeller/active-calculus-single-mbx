We see that: 1. For \(\int\,x\sin x\,dx\text{,}\) integration by parts is appropriate, because by taking \(u = x\) and \(v' = \sin(x)\text{,}\) we end up with an integral that is easy to find. 2. For \(\int\,{x^{4}\over 1 + x^{5}}\,dx\text{,}\) substitution is appropriate, because by taking \(w=1 + x^{5}\) we get an integral we can find. 3. For \(\int\,x^{4} e^{x^{5}}\,dx\text{,}\) substitution is appropriate, because by taking \(w = x^{5}\) we get an integral we can find. 4. For \(\int\,x^{4} \cos(x^{5})\,dx\text{,}\) substitution is appropriate, because by taking \(w = x^{5}\) we get an integral we can find. 5. For \(\int\,{1\over\sqrt{6 x + 1}}\,dx\text{,}\) substitution is appropriate, because by taking \(w = 6 x + 1\) we get an integral we can find.
