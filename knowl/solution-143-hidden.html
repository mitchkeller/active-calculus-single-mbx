SOLUTION
We first simplify the function
\begin{equation*} f(x) = \frac{4 x^2 \tan x }{ \sec x } = 4 x^2 \frac{\sin x}{\cos x}\cos x = 4 x^2 \sin x \end{equation*}
The product rule then gives
\begin{equation*} f'(x) = 4( x^2 \cos x + 2x \sin x ) \end{equation*}
Evaluating at \(x = 2\text{:}\)
\begin{equation*} f'(2) = 16 \cos(2) + 16 \sin(2) \end{equation*}
