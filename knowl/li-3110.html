Item 4.3.5.9.d.
Recall that the average value of a function \(f\) on an interval \([a,b]\) is
\begin{equation*} f_{\operatorname{AVG} [a,b]} = \frac{1}{b-a} \int_a^b f(x) dx\text{.} \end{equation*}
So the average velocity, \(AV\text{,}\) of the object on the interval \([0,4]\) is
\begin{equation*} AV = \frac{1}{4} \int_0^4 v(t) \, dt = (0.25)(-2.625) = -0.65625 \end{equation*}
feet per second.
