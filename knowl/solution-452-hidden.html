\(\int_0^{\pi/2}\,{1\over\cos^2(t)}\,dt\text{.}\) However, the integrand is undefined at \(t = \pi/2\text{,}\) so this is an improper integral. We find
\begin{equation*} \int_0^{\pi/2}\,{1\over\cos^2(t)}\,dt = \lim\limits_{a\to\pi/2^{-}}\,\int_0^a\,{1\over\cos^2(t)}\,dt = \lim\limits_{a\to\pi/2^{-}}\,(\tan(t))\bigg|_0^a = \lim\limits_{a\to\pi/2^{-}}( \tan(a) - \tan(0) ). \end{equation*}
However, \(\tan(a)\) is undefined as \(a\to\pi/2\text{,}\) so this integral diverges. Thus, because the integral diverges, the area is not bounded.
