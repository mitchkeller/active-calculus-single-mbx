Solution 6.5.5.4.1.

\(\int_0^{\pi/2}\,\tan(t)\,dt\text{.}\) However, the integrand is undefined at \(t = \pi/2\text{,}\) so this is an improper integral. We find
\begin{equation*} \int_0^{\pi/2}\,\tan(t)\,dt = \lim\limits_{a\to\pi/2^{-}}\,\int_0^a\,\tan(t)\,dt = \lim\limits_{a\to\pi/2^{-}}\,(-\ln|\cos(t)|)\bigg|_0^a = \lim\limits_{a\to\pi/2^{-}}( -\ln|\cos(a)| + \ln(1) ). \end{equation*}
However, \(-\ln|\cos(a)|\) is undefined as \(a\to\pi/2\text{,}\) so this integral diverges. Thus, because the integral diverges, the area is not bounded.
