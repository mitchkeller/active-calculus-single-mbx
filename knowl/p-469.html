We observe that \(|x+3| = (x+3)\) whenever \(x \ge -3\text{,}\) and that \(|x+3| = -(x+3)\) whenever \(x \lt -3\text{.}\) So, if \(x \gt -3\text{,}\) we see that
\begin{equation*} -\frac{|x+3|}{x+3} = -\frac{x+3}{x+3} = -1 \end{equation*}
whereas if \(x \lt -3\text{,}\) it follows that
\begin{equation*} -\frac{|x+3|}{x+3} = -\frac{-(x+3)}{x+3} = +1\text{.} \end{equation*}
Because the function value is \(-1\) to the right of \(-3\) and \(+1\) to the left of \(-3\text{,}\) it follows that the limit does not exist.
