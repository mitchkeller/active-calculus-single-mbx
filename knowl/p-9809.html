To compute \(E_{0.1}\) and \(E_{0.05}\text{,}\) we first compute the estimates to \(y(0.3)\) using \(\Delta t = 0.1\) and \(\Delta t = 0.5\text{.}\) Doing so and making similar assumptions about the error as we did in (a) and (b), we find that for \(\Delta t = 0.1\text{,}\) \(y(1) \approx 0.0290\) and for \(\Delta t = 0.05\text{,}\) \(y(1) \approx 0.0351\text{,}\) and therefore we consider the system
\begin{align*} y(0.3) - 0.0290 &= 0.1K\\ y(0.3) - 0.0351 &= 0.05K\text{.} \end{align*}
Solving the system, it follows that \(K = (-0.0290 + 0.0351)/0.05 = 0.122\) and \(y(0.3) = 0.1(0.122) + 0.0290 = 0.0412\text{,}\) which is much closer to the actual value of \(y(0.3) = 0.0408\text{.}\)
