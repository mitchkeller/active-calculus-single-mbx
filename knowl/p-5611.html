Now, this is one of our basic integrals because we can write it as \(\displaystyle \int -1.5 u^{-2} \, du\) and thus we can integrate it. Constants in front just stay multiplied so we get
\begin{equation*} \int -1.5 u^{-2} \, du = \frac{-1.5}{-1} u^{-1}+C \end{equation*}
Finally, make sure you put it back in terms of the original variable, which in this case is 'x'. So we substitute back in for 'u', which was \(\sec\!\left(2x\right)\text{.}\) Thus the final answer to this integral is
\begin{equation*} \int \frac{-3 \sec(2 x)\tan(2 x)}{\sec^{2}(2 x)} \, dx = 1.5\!\left(\sec\!\left(2x\right)\right)^{-1}+C \end{equation*}
