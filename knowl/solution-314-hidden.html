\(w = e^{3 x} + 1\text{.}\) Then \(dw = 3\,e^{3 x}\,dx\text{,}\) so that
\begin{equation*} \int\,\frac{e^{3 x}}{1 + e^{3 x}}\,dx = {1\over 3}\int\,{1\over w}\,dw = {1\over 3}\,\ln(|w|) + C = {1\over 3}\,\ln(|e^{3 x} + 1|) + C. \end{equation*}
Because \(e^{3 x} + 1\) is always positive, we can drop the absolute values from the argument of the natural log.
