Clearly the smallest \(x\) can be is 0 and the largest \(x\) can be is 5, since one side of the cardboard has length 10. But we're told in the problem to restrict the value of \(x\) to \(1 \le x \le 3\text{,}\) so this is the domain we use for \(V\text{,}\) even though \(V\) is defined for every real number \(x\text{.}\)
