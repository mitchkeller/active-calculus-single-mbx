  1. Let \(f(x) = \cos(x)\text{.}\)
    We know that \(f'(x) = -\sin(x)\text{,}\) \(f''(x) = -cos(x)\text{,}\) \(f'''(x) = \sin(x)\text{,}\) and \(f^{(4)}(x) = cos(x)\text{,}\) so \(f'\left(\frac{\pi}{2}\right) = -\sin\left(\frac{\pi}{2}\right) = -1\text{,}\) \(f''\left(\frac{\pi}{2}\right) = -cos\left(\frac{\pi}{2}\right) = 0\text{,}\) \(f'''\left(\frac{\pi}{2}\right) = \sin\left(\frac{\pi}{2}\right) = 1\text{,}\) and \(f^{(4)}(x) = cos\left(\frac{\pi}{2}\right) = 0\text{.}\) We also note that \(f\left(\frac{\pi}{2}\right) = \cos\left(\frac{\pi}{2}\right) = 0\)
    It follows that the degree four Taylor polynomial of \(f\) centered at \(x = \frac{\pi}{2}\) is
    \begin{equation*} T_4(x) = 0 - 1 \left(x - \frac{\pi}{2}\right) + \frac{0}{2!}\left(x - \frac{\pi}{2}\right)^2 + \frac{1}{3!}\left(x - \frac{\pi}{2}\right)^3 + \frac{0}{4!}\left(x - \frac{\pi}{2}\right)^4\text{,} \end{equation*}
    from which we can also easily determine \(T_3(x)\text{,}\) \(T_2(x)\text{,}\) and \(T_1(x)\text{.}\)
    From the pattern in \(T_4(x)\text{,}\) we find that the Taylor series for \(f(x)\) centered at \(x = \frac{\pi}{2}\) is
    \begin{equation*} T(x) = 0 - 1 \left(x - \frac{\pi}{2}\right) + \frac{0}{2!}\left(x - \frac{\pi}{2}\right)^2 + \frac{1}{3!}\left(x - \frac{\pi}{2}\right)^3 + \frac{0}{4!}\left(x - \frac{\pi}{2}\right)^4 + \cdots + (-1)^{n+1} \frac{1}{(2n+1)!}\left(x - \frac{\pi}{2}\right)^{2n+1} + \cdots \end{equation*}
    This result is not surprising, because \(f(x) = \cos(x) = -\sin\left( x - \frac{\pi}{2} \right)\text{.}\) So, if we stated with the Taylor series for \(g(x) = \sin(x)\) centered at \(a = 0\text{,}\) weâ€™d have
    \begin{equation*} g(x) = \sin(x) = \sum_{k=0}^{\infty} (-1)^{k} \frac{1}{(2k+1)!}x^{2k+1} \end{equation*}
    from which it follows that
    \begin{equation*} -\sin\left(x - \frac{\pi}{2} \right) = -\sum_{k=0}^{\infty} (-1)^{k} \frac{1}{(2k+1)!} \left(x - \frac{\pi}{2} \right)^{2k+1}\text{.} \end{equation*}
    Noting that \(-1 \cdot (-1)^k = -1^{k+1}\text{,}\) we have established that
    \begin{equation*} \cos(x) = -\sin\left(x - \frac{\pi}{2} \right) = \sum_{k=0}^{\infty} (-1)^{k+1} \frac{1}{(2k+1)!} \left(x - \frac{\pi}{2} \right)^{2k+1} \end{equation*}
    which matches our earlier result.

With \(f(x) = \frac{1}{1+x} = (1+x)^{-1}\text{,}\) we observe that \(f'(x) = -(1+x)^{-2}\text{,}\) \(f''(x) = (-1)(-2)(1+x)^{-3}\text{,}\) \(f'''(x) = (-1)(-2)(-3)(1+x)^{-4}\text{,}\) and \(f^{(4)}(x) = (-1)(-2)(-3)(-4)(1+x)^{-5}\text{.}\) Evaluating each of these functions at \(a = 1\text{,}\) we find \(f(1) = \frac{1}{2}\text{,}\) \(f'(1) = (-1) \cdot \frac{1}{2^2}\text{,}\) \(f''(1) = \frac{2!}{2^3}\text{,}\) \(f'''(1) = (-1) \cdot \frac{3!}{2^4}\text{,}\) and \(f^{(4)}(1) = \frac{4!}{2^5}\text{.}\)
    From these derivative values and the fact that \(c_k = \frac{f^{(k)}(1)}{k!}\text{,}\) it follows that
    \begin{equation*} T_4(x) = \frac{1}{2} - \frac{1}{2^2}(x-1) + \frac{1}{2^3}(x-1)^2 - \frac{1}{2^4}(x-1)^3 + \frac{1}{2^5}(x-1)^4\text{,} \end{equation*}
    from which we can also easily find \(T_3(x)\text{,}\) \(T_2(x)\text{,}\) and \(T_1(x)\text{.}\)
    Finally, from the patterns in our preceding work, we see the Taylor series for \(f(x)\) centered at \(a = 1\) is
    \begin{equation*} T(x) = \frac{1}{2} - \frac{1}{2^2}(x-1) + \frac{1}{2^3}(x-1)^2 - \frac{1}{2^4}(x-1)^3 + \frac{1}{2^5}(x-1)^4 + \cdots + \frac{1}{2^{n+1}}(x-1)^{n} + \cdots\text{.} \end{equation*}
