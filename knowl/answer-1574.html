Answer 8.5.2.1.

  1. \(g(x) = x^3 \sin(x^2) = x^5 - \frac{1}{3!}x^9 + \frac{1}{5!}x^{13} - \cdots \text{;}\) we expect this series to converge for all real numbers \(x\text{.}\)
  2. \(h(x) = e^{-x^4} = 1 - x^4 + \frac{1}{2!}x^8 - \frac{1}{3!}x^{12} + \cdots \text{;}\) the series converges for all real numbers \(x\text{.}\)
  3. \(p(x) = \frac{1}{1+5x} = 1 - 5x + (-5x)^2 + (-5x)^3 + \cdots = 1 - 5x + 5^2x^2 - 5^3x^3 + \cdots \text{;}\) the series converges for all \(x\) such that \(-\frac{1}{5} \lt x \lt \frac{1}{5}\text{.}\)
  4. \(q(x) = x^2 \ln(1+x^4) = x^6 - \frac{1}{2}x^{10} + \frac{1}{3}x^{14} - \cdots \text{;}\) the series converges when \(|x| \lt 1\text{.}\)
