Solution 1.3.3.5.1.

A. Recall that the derivative is positive when the function is increasing. So we’re looking for a point on the graph where the function is increasing; one such point is \(x = 1\text{.}\) B. The function is positive when the function is above the \(x\)-axis. One such point is \(x = 1.5\text{.}\) C. The derivative of the function is largest when the function is increasing the fastest. This occurs at \(x \approx 3.25\text{.}\) D. The derivative of the function is zero when the function has a horizontal tangent. One such point is \(x \approx 1.45\text{.}\) E. Finally, we’re looking for a point where the derivative is the same as the derivative at \(x = 0.75\text{.}\) This means that we want the slope of a tangent to the curve to be approximately the same as it is at \(x = 0.75\text{.}\) Such a point is \(x = 3.25\text{.}\)
