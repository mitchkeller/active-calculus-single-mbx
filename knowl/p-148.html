The table gives the amount of calories used per minute. So in order to find the total number of calories used in a half-hour, use the table to find the number of calories used by a 200 lb who walks for a half-hour, and then multiply by 30 minutes. Using the table we know a 200 lb person walking uses 5.4 calories per minute; therefore,

\begin{equation*} \text{Total calories used} = ( 5.4 )(30) = 162 \end{equation*}
