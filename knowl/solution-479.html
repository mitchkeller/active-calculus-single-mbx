Solution 7.6.4.4.1.

(a) A graph of \(dP/dt\) vs \(P\) is shown below.
Thus, \(dP/dt\) is negative when \(0 \lt P \lt b/a\) and positive when \(b/a \lt P \lt \infty\) (we ignore \(P \lt 0\text{,}\) because this doesn't make sense for a population). (b) We can see from the graph above that \(P\) is a decreasing function when \(P\lt \frac{b}{a}\text{.}\) Similarly, when \(P>\frac{b}{a}\text{,}\) the sign of \(dP/dt\) is positive, so \(P\) is an increasing function. Thus solution curves starting above \(\frac{b}{a}\) are increasing, and those starting below \(\frac{b}{a}\) are decreasing. We can also see that for \(P>\frac{b}{a}\text{,}\) the slope, \(\frac{dP}{dt}\text{,}\) increases with \(P\text{,}\) so the graph of \(P\) against \(t\) is concave up. For \(0\lt P\lt \frac{b}{a}\text{,}\) the value of \(P\) decreases with time. As \(P\) decreases, the slope \(\frac{dP}{dt}\) decreases for \(\frac{b}{2a}\lt P\lt \frac{b}{a}\text{,}\) and increases toward 0 for \(0\lt P\lt \frac{b}{2a}\text{.}\) Thus solution curves starting just below the threshold value of \(\frac{b}{a}\) are concave down for \(\frac{b}{2a}\lt P\lt \frac{b}{a}\) and concave up and asymptotic to the \(t\)-axis for \(0\lt P\lt \frac{b}{2a}\text{.}\) Thus, solution curves are similar to those shown in the graph below
\(P=\frac{b}{a}\) is called the threshold population because for populations greater than \(\frac{b}{a}\text{,}\) the population will increase without bound. For populations less than \(\frac{b}{a}\text{,}\) the population will go to zero, i.e. to extinction. The value \(\frac{b}{a}\) is an unstable equilibrium, and \(P=0\) is a stable equilibrium.
