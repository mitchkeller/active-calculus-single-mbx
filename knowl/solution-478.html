Solution 8.2.3.6.1.

  1. Since the ball bounces back \(3/4\) of the original height, we have

    \begin{equation*} h_1 = \left(\frac{3}{4}\right)h\text{.} \end{equation*}

  2. Since the ball bounces back \(3/4\) of the original height, we have

    \begin{equation*} h_2 = \left(\frac{3}{4}\right)h_1 = \left(\frac{3}{4}\right)^2h\text{.} \end{equation*}

  3. Since the ball bounces back \(3/4\) of the original height, we have

    \begin{equation*} h_3 = \left(\frac{3}{4}\right)h_2 = \left(\frac{3}{4}\right)^3h\text{.} \end{equation*}

  4. Since the ball bounces back \(3/4\) of the original height, we have

    \begin{equation*} h_n = \left(\frac{3}{4}\right)h_{n-1} = \left(\frac{3}{4}\right)^nh\text{.} \end{equation*}

  5. The distance traveled by the ball is

    \begin{align*} h+2[h_1+h_2+ \cdots ] \amp = h + 2 \left[ \left(\frac{3}{4}\right)^2h + \left(\frac{3}{4}\right)^2h + \left(\frac{3}{4}\right)^2h + \cdots \right]\\ \amp = h + 2\left(\frac{3}{4}\right)h \left[1 + \left(\frac{3}{4}\right) + \left(\frac{3}{4}\right)^2 + \cdots \right]\\ \amp = h + \left(\frac{3}{2}\right)h \sum_{n = 0}^{\infty} \left(\frac{3}{4}\right)^n\\ \amp = h + \left(\frac{3}{2}\right)h \left(\frac{1}{1-3/4}\right)\\ \amp = h + \left(\frac{3}{2}\right)h (4)\\ \amp = h + 6h\\ \amp = 7h\text{.} \end{align*}

    So no matter what the initial height of the ball, the ball will bounce infinitely many times but only travel a finite distance.
