Solution 9.3.2.8.1.

\begin{equation*} \frac{2x}{x^2-4}=\frac{A}{x+2}+\frac{B}{x-2} \end{equation*}
so that
\begin{equation*} 2x = A(x-2)+B(x+2) = (A+B)x + 2 (-A+B). \end{equation*}
Solving, we get \(A=B=1\text{,}\) so that
\begin{equation*} \int \frac{2x}{x^2-4}\,dx= \int \left(\frac{1}{x+2}+\frac{1}{x-2}\right)\,dx =\ln|x+2|+\ln|x-2|+C. \end{equation*}
Using the substitution \(w=x^2-4\text{,}\) we get \(dw= 2x\,dx\text{,}\) so we have
\begin{equation*} \int \frac{2x}{x^2-4}\,dx= \int \frac{dw}{w}=\ln|w|+C = \ln|x^2-4|+C. \end{equation*}
The properties of logarithms show that the two results are the same:
\begin{equation*} \ln|x+2|+\ln|x-2| = \ln|(x+2)(x-2)| =\ln|x^2-4|. \end{equation*}
