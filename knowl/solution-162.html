Solution 2.6.6.5.1.

\((2,6)\) is on the curve, we know \(f(2)=6\text{.}\) The point \((2.1,6.2)\) is on the tangent line, so
\begin{equation*} \hbox{Slope of tangent} = \frac{6.2-6}{2.1-2} = \frac{0.2}{0.1} = 2. \end{equation*}
Thus, \(f'(2)=2\text{.}\) By the chain rule
\begin{equation*} h'(2) = 5(f(2))^{4}\cdot f'(2) =5\cdot 6^{4}\cdot 2 = 12960. \end{equation*}
Because \(g(x)\) is the inverse function of \(f\text{,}\) we know that \(g(6) = 2\text{.}\) Then note that \(f(g(x)) = f(f^{-1}(x)) = x\text{,}\) so \(f(g(x)) = x\text{.}\) Differentiating both sides of the equation, \(f'(g(x))\cdot g'(x) = 1\text{,}\) so \(g'(x) = \frac{1}{f'(g(x))}\text{.}\) Thus \(g'(6) = \frac{1}{f'(g(6))} = \frac{1}{f'(2)} = \frac1{2}\text{.}\)
