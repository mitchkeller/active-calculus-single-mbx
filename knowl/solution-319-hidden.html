A. \(\int\,7\,x^{2}\,(x^{3} + 1)\,dx = \int\,7\,x^{5} + 7\,x^{2}\,dx\text{.}\) Thus, antidifferentiating term-by-term, we get \(\int\,7\,x^{2}\,(x^{3} + 1)\,dx = {7\over6}\,x^{6} + {7\over3}\,x^{3} + C\text{.}\) B. \(\int\,7\,x^{2}\,(x^{3} + 1)\,dx = \int\,{7\over3}\,w\,dw\text{.}\) Thus, antidifferentiating, we get \(\int\,7\,x^{2}\,(x^{3} + 1)\,dx = {7\over6}\,w^2 + C = {7\over6}\,(x^{3} + 1)^2 + C\text{.}\) C. To see the difference between these two answers, we expand the second: \((x^{3} + 1)^2 = x^{6} + 2\,x^{3} + 1\text{,}\) so the difference is
\begin{equation*} ({7\over6} (x^{6} + 2\,x^{3} + 1)) - ({7\over6} \,x^{6} + {7\over3}\,x^{3}) = {7\over6}. \end{equation*}
That is, the difference is a constant! This is completely reasonable: the antiderivative is only determined up to a constant, and the arbitrary constant \(C\) that we get for the most general antiderivative will adjust for this difference.
