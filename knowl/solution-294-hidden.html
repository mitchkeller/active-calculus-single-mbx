    • \(\displaystyle s(1) = s(0) + \int_0^1 v(t) \, dt = \frac{1}{2} + \frac{7}{6} = \frac{5}{3}\)

    • \(\displaystyle s(3) = s(1) + \int_1^3 v(t) \, dt = \frac{5}{3} - \frac{8}{3} = -1\)

    • \(\displaystyle s(5) = s(3) + \int_3^5 v(t) \, dt = -1 - \frac{8}{3} = -\frac{11}{3}\)

    • \(\displaystyle s(6) = s(5) + \int_5^6 v(t) \, dt = -\frac{11}{3} + \frac{7}{6} = -\frac{5}{2}\)

  2. The position function \(s\) is increasing on \(0 \lt t \lt 1\) and \(5 \lt t \lt 6\) since this is where \(v(t) = s'(t)\) is positive. Similarly, \(s\) is decreasing for \(1 \lt t \lt 5\text{.}\)

  3. The position function \(s\) is concave down for \(t \lt 2\) since \(v = s'\) is decreasing on this interval, and \(s\) is concave up for \(t \gt 2\) because \(v = s'\) increases there.
